# 内容资产区

本目录用于存放已经发布、准备发布或需要长期沉淀的平台内容资产。

---

## 1. 目录定位

这里放的是内容成品与平台素材，不是正式营销规则文档。

适合放入本目录的内容：
1. 已发布的微信公众号文章
2. 已发布或待发布的 X 帖子、线程
3. Reddit 帖子、评论素材、AMA 草稿
4. Product Hunt 上线文案、Maker Comment、FAQ
5. 官网 Blog 正文草稿与已发布归档

---

## 2. 不应放入本目录的内容

以下内容不应放在这里：
1. 营销框架
2. 社媒运营规范
3. SEO / GEO 执行规范
4. 项目规划
5. 账号授权、数据治理、合同条款

这些内容应进入 `docs/`。

---

## 3. 判断标准

判断一个文档是否进入本目录，按以下顺序：
1. 如果它回答的是“怎么运营、谁负责、按什么规则执行”，放 `docs/03-marketing/`
2. 如果它回答的是“账号权限、支付、合同、授权、风险边界”，放 `docs/05-governance/`
3. 如果它本身就是要发出去的内容成品或素材资产，放本目录

快速规则：
1. `规则文档` 进 `docs/`
2. `内容成品` 进 `内容资产/`
