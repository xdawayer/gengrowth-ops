# 配图 Prompt

## 1. 封面图

```
微信公众号封面图，2.35:1比例，无任何文字。
延续GenGrowth成长笔记统一视觉风格（深石墨灰背景）。
构图：左侧手绘散乱元素（占30%）→ 中央发光焦点（占40%）→ 右侧有序图标（占30%）。
左侧画面：手绘风格的碎片化输入——几个大小不一的聊天气泡漂浮散落，气泡内有模糊文字痕迹；旁边几张手绘便签纸和文档碎片，带轻微粉笔质感，表达"散落在对话里、没人承接的输出"。
中央焦点：一个圆形汇聚漩涡，外圈是手绘线条，内核是琥珀金/暖白发光效果，多条细线从左侧碎片汇聚进入圆心，象征"正式工作被系统接住"的转换时刻，光晕清晰，在深色背景上形成强视觉焦点。
右侧输出：从漩涡右侧射出3条线，分别连接：一个文档叠加图标（知识库）、一个分支图标（版本链路）、一个终端光标图标（执行入口），手绘风格，米白线条，图标之间有充足间距。
整体无任何文字，不写任何字符。
整体传达：从散乱无序的AI对话，进入一个能被管理、被追踪、被复用的系统——那个"接住"的瞬间。
```

**用途**：文章封面
**尺寸**：2560×1088

## 2. 正文插图：四层递进架构图

```
微信公众号正文插图，16:9比例，无任何文字。
延续GenGrowth成长笔记统一视觉风格（深石墨灰背景，高级手绘战略图风格）。
主题：小团队AI主工作流的四层承接系统。
画面核心：
深石墨灰背景上，四个水平排列的手绘圆角矩形，从左到右依次为：
第一个矩形内有一个文档叠加图标，下方米白小字"上下文"；
第二个矩形内有一个Git分支图标（Y形三叉分支，不是USB符号），下方米白小字"版本"；
第三个矩形内有一个终端命令行光标图标（>_），下方米白小字"执行"；
第四个矩形内有一个齿轮图标，下方米白小字"流程边界"。
矩形之间用带箭头的米白连线串联，每条连线上方有一个手绘问号气泡，暗示每层回答上一层引出的问题。
底部一条细米白横线贯穿，下方米白小字"正式工作的承接链路"。
配色：矩形边框和图标用低饱和青蓝、淡青绿、橙黄、米白各一个区分，背景保持深石墨灰。
```

**用途**：放在第五节和第六节之间（四层拆解完毕、进入PRD链路之前）
**尺寸**：1920×1080
