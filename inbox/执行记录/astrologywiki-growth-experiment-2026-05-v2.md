## B.0 实施背景：学习实验 vs 证明实验

> 本附录的修改建议以"对外输出可信案例"为严谨度标准。种子阶段适用**渐进式严谨度**，不一次性落地所有项目。

**种子阶段的定位是学习实验，不是证明实验：**

|  | 学习实验（当前）| 证明实验（未来）|
|---|---|---|
| 核心目标 | 快速知道什么有效，快速调整 | 向外部证明因果关系 |
| 严谨度 | 记录执行动作 + 时间点 + 数据快照 | 完整归因链 + 可重复性 |
| 控制组 / 预注册 | **不需要** | 对外案例时推荐 |

**渐进式严谨度路线图：**

| 客户轮次 | 核心要求 | 本附录对应 |
|---------|---------|----------|
| 第 1 个客户 | 记录做了什么、发生了什么、时间戳 | **B.3.0**（Week 1-2 聚焦）+ **B.3.1**（核心假设表）|

## B.1 核心定性（三方共识）

1. **方向对**：SEO + Social Probe 双轨适合早期种子客户，没有方向性错误。
2. **严谨度不足**：缺反事实、缺控制组、缺干预日志、缺品牌/非品牌拆分、缺成本记录。
3. **60 天 ≠ SEO 可验证周期**：Ahrefs 自身研究指出 SEO 通常 3-6 个月起效；60 天只能验证领先指标（收录、impressions、Top 50/20 进入率），不能稳定证明商业转化。
4. **2026 年 SEO 新现实未纳入**：AI Overviews 普遍部署、零点击搜索增加、HARO 已被 Featured.com 收购、Core Web Vitals 已用 INP 替代 FID——这些 v1.2 都未反映。
5. **过度依赖 KD**：`KD < 20` 是第三方估算，不是 Google 难度；要叠加 SERP 弱度、AIO 风险、点击潜力、页面匹配度。

**执行优先顺序：**

2. **内容创作与发布**（Week 1 Day 3–5，主要工作量）

3. **收录追踪**（Week 2，每天 5 分钟）

### B.3.3 双轨贡献分析改为「证据等级归因」

| 等级 | 标准 | 可宣称 |
|---|---|---|
| **A 级** | 有 UTM / GA4 key event / CRM source 直接追踪 | "X 转化由该动作带来" |

> 对外案例只允许 A 级和 B 级出现"百分比"，C 级写"观察到"。

### B.3.4 Week 3 检查点改为决策门（三选一）

替换原文「非阻塞」逻辑：

```markdown
#### Week 3 决策门（必须输出三选一）
- [ ] **双轨继续**：两轨均有领先指标信号（SEO 收录 + Social 有效互动）
- [ ] **仅 Track A 继续**：Social Probe 无信号但产品方向已验证；Track A 单独跑完 60 天
- [ ] **双轨暂停**：Social 无信号 + SEO 收录率 < 50% + 技术问题未解决；先做诊断
```

### B.3.5 外链建设：删数量目标，改质量管线

**删除：**
- 「目录/工具聚合站 10-15 个」
- 「Guest post 1-2 篇」（除非有真实编辑价值）
- 「HARO」（已停服）

**替换为：**
```markdown
| 类型 | 60 天目标 | 质量门槛 |

| 高相关引用 | 3-5 个 | 主题相关 + DR ≥ 20 + 真实流量 |

| 品牌提及转链 | 持续处理 | Ahrefs Alerts 监控 |

| 站点 | 相关性 | 目标 URL | Link Type | dofollow/nofollow | Anchor | 上线日期 | 是否收录 | 风险等级 |
```

## B.4 P1 详细方案

### B.4.1 关键词优先级新公式

```
Priority = 商业意图 + SERP 弱度 + 点击潜力 + 页面匹配度 + Social 证据 + 趋势性
         - AIO/零点击风险 - 执行成本

KD 仅作输入参考，不作分层规则。
```

「Tier 1 快速胜利」的新定义：
> 低 KD + 明确意图 + SERP 弱（Top10 有 UGC/论坛/薄内容）+ 无强 AIO 吞点击 + 7 天内可产强页 + 有内链入口

### B.4.2 新增「旧内容刷新」工作流

> 60 天内 ROI 最高的动作之一：刷新已位列 Page 2 (#11-20) 的旧页面，比从 0 写新页快得多。v1.2 完全缺失这一节。

```markdown
#### 旧内容刷新（Week 1 启动，与新内容并行）
| URL | 当前排名 | 目标关键词 | 刷新动作 | 完成日期 | W4 排名 | W8 排名 |
```

### B.4.3 竞品同期对照（Day 0 + Day 60 都要记录）

```markdown
### 竞品同期可见度
| 竞品 | 自然流量估算 | Top 排名词数 | 新增/失去关键词 | 备注 |
|------|----------|-----------|--------------|-----|
| 竞品 A | | | | |
| 竞品 B | | | | |
```

> 没有这个，Day 60 的「+30% 流量」可能只是行业季节性，归因近乎为 0。

### B.4.4 「→ SEO 转化？」判断标准

```markdown
> **判断为「是」的标准（满足任一即可）：**
> - 评论中出现 ≥ 3 次相同长尾表达 → 该词进 Track A 候选池
> - 单贴互动 > 平台均值 2× → 该 angle 进 SEO 内容计划
> - 帖子被竞品/媒体引用 → 该话题列入 Tier 1 趋势抢占
```

### B.4.5 60 天指标三层分拆

| 中期指标 | Day 30 | 非品牌 impressions、目标 query 进入 Top 50/Top 20 数量、CTR、organic engaged sessions |
| 业务结果 | Day 60 | 非品牌 clicks、organic key events、注册/留资质量、Social UTM 转化、是否值得继续投入 |
