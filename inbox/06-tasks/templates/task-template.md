# Task 单任务模板

## 用法

1. 复制本文件到 `inbox/06-tasks/tasks/`。
2. 按 `YYYY-MM-DD-<topic>-task.md` 重命名。
3. 填写一条主任务和下面各字段。

## 模板正文

```markdown
---
title: <任务标题>
date: YYYY-MM-DD
updated: YYYY-MM-DD
type: note
tags:
  - task
  - <主题标签>
aliases:
  - <可选别名>
---

# <任务标题>

- [ ] [<模块>] <任务描述> #task #owner/<name> 📅 YYYY-MM-DD

## 目标

- ...

## 验收标准

- ...

## 来源链接

- ...

## 执行记录

- YYYY-MM-DD：任务创建
```
