### 2.2 待采购外链规格要求

每条待采购的外链必须同时满足以下标准，缺一不可：

| 指标 | 要求 | 说明 |
|------|------|------|
| Ahrefs DR | **45-75** | 低于45权重贡献有限；高于75价格骤升 |
| 均分DR | **≥ 0.04** | DR ÷ Ahrefs dofollow referring domains，低于此值权重被严重稀释 |
| 有机流量 | **≥ 1,000/月（Ahrefs）** | 确认非零流量站，流量为0是PBN特征 |
| 流量国家 | **英语国家为主** | US/UK/AU/CA占比 > 50% |
| 链接类型 | **dofollow，书面确认** | 付款前明确写入邮件往来记录 |
| TAT | **≤ 10天** | 付款前书面确认上线时间 |
| 内容方向 | 灵性/健康/女性生活/自我成长/心理学 | 话题相关性提升链接价值 |
| 排除特征 | 站龄 < 1年 + DR高速增长 | 新站DR在几个月内从0飙升至50+是PBN信号 |

**均分DR查询方法：**
```
ahrefs.com/backlink-checker → 输入域名 → 查看"Dofollow"数量
均分DR = Ahrefs DR ÷ Dofollow referring domains数量
```

### 4.2 采购优先顺序

```
第一优先：DR 55-75 + 均分DR ≥ 0.05 + 灵性/健康/女性内容
第二优先：DR 45-55 + 均分DR ≥ 0.04 + 有真实英语国家流量
不考虑：均分DR < 0.04 的任何站点，无论DR多高
```

### 4.3 审核流程（每个候选站点）

```
Step 1：ahrefs.com/backlink-checker → 查DR和Dofollow RDs数量
Step 2：计算均分DR = DR ÷ Dofollow RDs → 低于0.04直接排除
Step 3：确认Ahrefs月流量 > 1,000 且英语国家为主
Step 4：站龄核查（WHOIS）→ 建站不足1年且DR > 40需警惕
Step 5：邮件确认dofollow属性 + TAT ≤ 10天 → 付款
```
