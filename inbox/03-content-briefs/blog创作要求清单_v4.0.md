# GenGrowth SEO 博客创作终极要求清单 (v4.0 准则)

本清单汇集了从 v2.2 到 v4.0 迭代中的所有核心共识，是审核 AI 产出内容的最高裁决标准。其核心目标是平衡 **Google 搜索排名（SEO）**、**AI 引擎引用（GEO）** 以及 **专家级品牌权威（EEAT）**。

---

## 1. 标题与摘要锁定 (SEO & Snippet Lock)
- [ ] **H1 磁性标题**：关键词必须自然融入标题前 60 个字符内（最好在前 3-5 个词）。严禁使用 `[关键词]: [从句]` 这种死板模板。
- [ ] **Snippet 锁定（双句法）**：H1 下的第一段首句必须是加粗的标准定义句（`[Target_Keyword]` is...）。
- [ ] **功能总结**：定义句后紧跟核心功能说明，并配以 3 个项目符号（Bullet Points）总结核心特征。

## 2. 链接阶梯与权重地图 (Link Master)
- [ ] **数量阶梯控制**：
    - T1 (1500-1800w): 5 个内链 (1 Pillar, 3 Spokes, 1 CTA)。
    - T2 (1000-1200w): 3 个内链 (1 Pillar, 1 Spoke, 1 CTA)。
    - T3 (600-800w): 1-2 个内链 (1 Pillar, 1 CTA)。
- [ ] **首链优先权**：若是 Spoke 页面，必须在开篇 150 字内回链 Pillar 页面。
- [ ] **语义锚文本**：严禁使用“点击这里”或单纯的关键词。锚文本必须包含：`目标词 | 语义背景 | 点击收益`。
- [ ] **外部权威背书**：T1/T2 必须包含 1-2 个指向 Wikipedia、NASA 或学术机构的外部链接，且必须带 `target="_blank"`。

## 3. 原子化知识架构 (Atomic GEO Layout)
- [ ] **隐形原子块**：每个 H2 必须包含“事实金句（Topic Sentence） -> 逻辑/机制 -> 实例/结果”的三段式结构。
- [ ] **结构标签封印**：严禁在正文中输出“Topic Sentence”或“Example”等引导标签。
- [ ] **段落控制**：任何段落不得超过 4 行。
- [ ] **决策型表格**：表格必须包含“如何观察”或“应用场景”列，单元格内必须是完整的短句。

## 4. 实体图谱与系统背景 (Entity & Systemic Context)
- [ ] **实体三角拓扑**：必须明确描述 `核心实体 ↔ 关联主宰 ↔ 对应特征` 的三角关系。
- [ ] **系统性对比**：禁止孤立解释。必须将该实体与其“功能对立面”或“系统下一阶段”进行对比（如 9 宫对比 3 宫）。
- [ ] **权威实证锚定**：必须在正文叙事中自然提及领域内的权威创始人或学派（如 Dane Rudhyar 等），增加 EEAT 信任感。

## 5. 语言边界与去 AI 化 (No-AI Filter)
- [ ] **去人格化**：严禁出现“你的逻辑”、“如你所说”、“这让你感觉”等私人对话痕迹。统一改为“专家观测到”、“此配置通常关联”等客观叙述。
- [ ] **动词封印**：严禁使用 `is about` / `relates to` 等弱动词。强制使用 `governs` (掌管) / `filters` (过滤) / `modulates` (调制) / `correlates with` (关联)。
- [ ] **术语禁令**：禁止使用 `recursive` (递归)、`mechanism` (机制)、`architecture` (架构) 等 AI 味重的学术词汇。
- [ ] **科学边界**：严禁在占星等领域将解释与“科学研究/证明”挂钩。

## 6. 视觉资产与图片 SEO (Visual Assets & Image SEO)
- [ ] **内容匹配度**：每篇 blog 必须包含至少 1 张与核心内容或核心实体（Primary Entity）高度匹配的插图，用于辅助理解复杂概念。
- [ ] **文件格式与优化**：首选 **WebP** 格式以平衡清晰度与加载速度；禁止直接上传超过 200KB 的大图。
- [ ] **语义化命名**：图片文件名必须采用与关键词相关的英文小写及连字符（kebab-case）。例：`9th-house-astrology-chart.webp`；严禁使用 `IMG_123.jpg` 等无意义命名。
- [ ] **Alt Text 必填**：每张图片必须配置描述性的 Alt Text，自然融入目标关键词，用于提升图片搜索排名及无障碍访问体验。

## 7. 转化与收尾 (Conversion & FAQ)
- [ ] **实战型 FAQ**：禁止问傻瓜问题。必须回答用户在真实操作中遇到的“摩擦点”和“矛盾点”（PAA 驱动）。
- [ ] **决策驱动 CTA**：CTA 必须符合 `动作 -> 产出 -> 收益` 公式。
- [ ] **品牌灵魂沉淀**：Reflection Prompts（反思问题）必须下沉到文章最末尾，确保不稀释上文的语义密度。
