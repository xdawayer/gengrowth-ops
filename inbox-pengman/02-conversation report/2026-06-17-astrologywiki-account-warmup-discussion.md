# AstrologyWiki 账号养号判断与执行建议

> 记录我们关于“新号直接发视频是否影响流量、是否需要养号、接下来怎么做”的讨论结论。

## 1. 当前结论

账号资产表已经进入阶段性完成状态，但还不是终点；仍建议补齐：

- 2FA 状态
- 恢复方式
- 主页链接策略

关于“新注册就直接发视频”的影响，结论是：

> **有关系，但通常不是唯一原因，也不一定是主因。**

更可能是以下因素叠加：

1. 新号冷启动；
2. 账号资料未完全补齐；
3. 账号没有互动历史；
4. 首条内容的主题 / 包装 / 平台适配度；
5. 平台对首批内容的初始测试反馈。

## 2. 分平台判断

### 2.1 X

- X 对账号真实性、质量和安全排名比较敏感。
- 新号直接发内容，尤其是带链接、带品牌属性、缺少互动历史时，初始分发可能偏弱。
- **养号有帮助，而且对早期推荐可能比较明显。**

### 2.2 TikTok

- TikTok 的 For You feed 主要看用户互动、内容信息和用户信息。
- 账号是否有稳定的互动历史，会影响系统如何判断内容归属。
- **养号有帮助，能改善冷启动质量。**

### 2.3 YouTube / Shorts

- YouTube 更看重观众行为：观看、搜索、订阅、点赞、喜欢/不喜欢、"Not interested" 等。
- 账号新不新不是唯一关键，内容质量、标题、封面、留存和主题匹配更重要。
- **养号有一点帮助，但不是决定性因素。**

## 3. 养号的定位

养号不是“流量开关”，更像是：

- 给平台建立正常行为历史；
- 让平台更容易理解账号属于什么内容圈；
- 降低新号直发的陌生感；
- 为后续内容分发打底。

## 4. 推荐的轻量养号动作

### 第 1 步：补齐 profile

- 头像
- banner
- 简介
- 链接
- 账号一致性

### 第 2 步：连续 3–7 天自然互动

- 浏览相关内容
- 少量点赞 / 收藏
- 关注同类账号
- 偶尔自然评论
- 避免一上来高频导流

### 第 3 步：发低风险内容

优先发：

- 占星基础解释
- 短观点型内容
- 轻量图文 / 短视频
- 不强推导流

### 第 4 步：观察平台反馈

重点看：

- 是否有基础曝光；
- 是否有评论 / 收藏 / 转发；
- 哪个平台更容易有互动；
- 哪个平台更适合 AstrologyWiki 的表达。

## 5. 对当前视频结果的判断

梅西视频的低流量，不应简单归因为“没养号”。

更合理的判断是：

- 新号冷启动有影响；
- 账号结构还不完整；
- 内容与平台匹配度需要继续观察；
- 首条内容本身仍在测试阶段。

## 6. 接下来可执行的动作

1. 完成账号资产的最后补齐项；
2. 做 3–7 天轻量养号；
3. 继续发第二条常规型内容；
4. 记录每个平台的结果；
5. 把“账号资产 → 内容生产 → 发布 → 复盘”沉淀成 SOP。
