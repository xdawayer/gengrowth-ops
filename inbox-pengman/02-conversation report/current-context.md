- Pengman 可以参与部分主题词和用户问题调研，但不是关键词体系负责人。

- Pengman 发布的平台内容若包含 AstrologyWiki 链接，属于内容分发产生的自然外部链接和引荐流量；不等同于系统性的 SEO 外链建设。

## 3. 当前已知决定

- 项目整体目标是冲刺 AstrologyWiki  PV 5,000、美国流量为主；这是整体目标，不应直接视为 Pengman 单独负责的数据 KPI。
- 当前了解到 AstrologyWiki 的平台账号可能都需要从零建立。
- 注册前仍应确认是否存在未交接的历史账号，避免重复注册。
- 当前工作顺序：调研 → 制定方案并确定平台与平台内容 → 注册账号 → 养号 → 初步测试。
- YouTube 已确定必做，属于核心平台。主要价值是视频搜索覆盖、Google 搜索可见性、品牌与主题权威、长期内容资产和引荐流量。
- 暂无官方依据证明“因为 YouTube 属于 Google，所以指向 AstrologyWiki 的链接会自动获得额外排名权重”；不应把 YouTube 简化为高权重外链。
- 必要时可参考或试用“社媒自动化营销竞品概览”和“增长自动化 Top 5”中的工具。它们并非全部是 GenGrowth 直接竞品，而是包括局部直接竞品、相邻竞品、底层平台和渠道工具。
- 社媒 / 社区内容除导流外，也承担用户研究和主题验证作用。
- 内容应保持 AstrologyWiki 的现代、反宿命、自我理解和心理安全边界。
