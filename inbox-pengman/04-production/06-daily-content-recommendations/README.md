# Daily Content Recommendations

Store generated weekday recommendations here.

Suggested file name:

```text
YYYY-MM-DD-daily-content-recommendation.md
```

Each note should include:

- today's primary recommendation
- one ready-to-post draft or short-video script
- 2-3 backup ideas
- linked AstrologyWiki article
- GSC evidence used, if available
- recent published topics excluded
