# 参考资料区

本目录用于存放 `gengrowth-wiki` 根目录下的非正式参考资料。

---

## 1. 目录定位

这里的内容默认不属于正式文档主库，不进入 `docs/`。

它们通常属于以下类型：
1. 外部学习资料
2. 阅读输入
3. 参考案例
4. 临时研究材料

---

## 2. 包含内容

当前本目录主要收纳：
1. AI 与 Agent 学习资料
2. 外部技术原理拆解
3. 外部案例文章
4. 方法论参考材料
5. 工具类说明、安装指引与分析文档

例如可集中存放在 `tool-guides/`。

---

## 3. 不包含内容

以下内容不应放入本目录：
1. 商业计划书
2. 项目规划
3. PRD、SPEC、设计稿、评审
4. 营销框架与执行规范
5. 合同、授权、治理规则

这些内容都必须进入 `docs/`。

---

## 4. 使用规则

1. 本目录内容默认作为输入与参考，不作为正式执行依据。
2. 若某份内容被长期引用、反复维护或升级为标准文档，应迁入 `docs/`。
3. 根目录仅保留 `AGENTS.md`、`REFERENCE.md` 与少量工作台目录；参考类 Markdown 优先放入本目录。
