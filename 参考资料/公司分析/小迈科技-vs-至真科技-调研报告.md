# 小迈科技 vs 至真科技 对比调研报告

> 报告日期：2026-03-06 | 调研目的：海外业务发展情况对比，聚焦规模、业务线与商业模式

---

## 一、执行摘要

小迈科技和至真科技均于 2015 年 1 月在广州天河区成立，同属移动互联网出海赛道，但经过十余年发展已分化为两种截然不同的商业模式。

**小迈科技**定位"开发者孵化运营平台"，以自研工具 APP 和休闲游戏为核心，通过广告变现（IAA）模式积累了 **7 亿+累计下载、500-1000 万 DAU**，走"批量化产品开发 → 流量获取 → 广告变现"的纵向一体化路线。创始人梁国盛出身久邦数码（NASDAQ 上市公司），擅长移动产品规模化运营。

**至真科技**定位"企业出海数字服务商"，从程序化广告起步（前身 Batmobi），发展出 **NetMarvel（效果营销平台）+ 超店 Shoplus（跨境电商 SaaS）+ 点金石 GameGoing（游戏研发发行）**三大业务线，覆盖 **230+ 国家、12 亿+ MAU、月广告请求 3000 亿+、年 GMV 超 20 亿**，走"广告技术 + 电商 SaaS + 游戏发行"的横向平台化路线。

**关键发现：**
- 至真科技的海外流量规模（12 亿 MAU、3000 亿+月广告请求）**约为小迈科技的 10-20 倍**，且收入结构更多元化（广告技术服务费 + SaaS 订阅 + 游戏发行分成）
- 小迈科技更偏"产品型公司"（自研 APP 矩阵 + 广告变现），至真科技更偏"平台型公司"（服务 2000+ 广告主 + 电商商家 + 游戏开发者）
- 两家公司均面临 Apple ATT 隐私政策冲击（ATT opt-in 率已从 2021 年的 26% 降至 2024 年的不足 14%），但至真科技凭借多引擎布局，抗风险能力更强
- 至真科技 2025 年再获广州"未来独角兽"称号，并以 NetMarvel + 点金石双品牌参展 ChinaJoy；小迈科技 2022 年入选拟上市高企百强后，近两年公开动态较少

## 三、业务与产品分析

### 3.1 产品/服务矩阵对比

| 维度 | 小迈科技 | 至真科技 |
|------|---------|---------|
| **业务线数量** | 2 条（自有产品 + 广告系统） | 3 条（NetMarvel + Shoplus + GameGoing） |
| **核心产品** | 自研 APP 矩阵（工具+游戏） | NetMarvel 效果营销平台 |
| **产品数量** | 累计 400+ 款，核心运营 10 余款 | 500+ 自有产品 + 平台服务 |
| **广告技术** | 聚量（投放）+ 洪流（变现） | SSP / DSP / ADX 程序化广告全栈 |
| **电商业务** | 无 | 超店 Shoplus（独立站 SaaS） |
| **游戏业务** | 自研休闲游戏（IAA 变现） | 点金石 GameGoing（研发+发行+代投） |
| **数据系统** | 天眼（分析）+ 天启（赋能） | AI 大数据营销系统 |

#### 小迈科技产品矩阵详解

1. **工具 APP**（核心流量来源）
   - 清理类：清理星等——手机垃圾清理、内存加速
   - 充电类：充电得宝等——电池管理、充电动画
   - 车主类：车行无忧——违章查询、保养提醒
   - **问题：** 2021 年工信部通报清理星（违规收集个人信息）和充电得宝（强制定向推送），暴露了**用户隐私合规风险**。该类工具 APP 的变现高度依赖广告展示频次，容易与用户体验产生矛盾（[新浪财经报道](https://finance.sina.com.cn/jjxw/2021-04-21/doc-ikmxzfmk8049159.shtml)）

2. **休闲游戏**（增长引擎）
   - 以 IAA 模式为主，轻研发、重买量
   - 品类覆盖消除、跑酷、益智、运动模拟等
   - 核心策略：快速立项 → 买量测试 → 达标则放量、不达标则放弃

3. **广告系统**（变现基础设施）
   - **聚量**：自有产品的广告投放系统，优化 CPI/ROAS
   - **洪流**：广告变现 SDK，聚合多家 Ad Network（AdMob、AppLovin 等）
   - **天眼**：用户行为分析系统，支撑精细化运营
   - **天启**：开发者赋能工具，提供选品、数据看板等

4. **开发者孵化**
   - 模式：小迈提供资金、流量和技术，开发者/小团队贡献创意和执行
   - 实质：更像内部工作室/项目制，而非开放平台
   - 规模：对外服务的开发者数量未公开

#### 至真科技产品矩阵详解

1. **NetMarvel 效果营销平台**（收入主引擎）
   - **前身**：Batmobi（2015-2021），2021 年品牌升级为 NetMarvel
   - **定位**：全球一站式效果营销服务平台
   - **技术栈**：自建 SSP（供给侧平台）+ DSP（需求侧平台）+ ADX（广告交易平台），形成程序化广告全栈能力
   - **流量规模**：月广告请求 3000 亿+，覆盖 230+ 国家，12 亿+ MAU
   - **流量来源**：自有 500+ 产品矩阵 + 100 万+ SDK 合作应用
   - **服务能力**：CPI/CPA 效果付费、品牌广告、再营销
   - **合作伙伴**：Facebook、TikTok、Google 等头部媒体
   - **服务客户**：2000+ 广告主

2. **超店 Shoplus 跨境电商 SaaS**（第二增长曲线）
   - **定位**：跨境电商独立站 SaaS 建站平台
   - **子产品线**：
     - **超店建站**：一键建站，100+ 跨品类模板（时尚、美妆、家居、母婴、户外等）
     - **超店有数**：TikTok 电商选品与营销分析平台
   - **核心功能**：多语言/多货币展示、PayPal/Stripe/信用卡等国际支付、优惠券/限时折扣/捆绑销售、Meta/Google/TikTok 广告对接、SEO 优化、邮件营销
   - **定价**：月费 <$20 起步，交易佣金 0.5%——**显著低于 Shopify（$39/月起，佣金 2%）和店匠 Shoplazza（$28/月起）**
   - **获客策略**："百万扶持计划"（0 成本开店）
   - **年 GMV**：超 20 亿元（据公开报道）

3. **点金石 GameGoing 游戏业务**（第三增长曲线）
   - **定位**：集研发、发行、代投于一体的游戏出海综合服务商
   - **核心能力**：自研休闲/超休闲游戏 + 代理发行 + 投放代运营
   - **与 NetMarvel 的协同**：利用 NetMarvel 的广告流量数据 + GameGoing 的用户数据资产，实现精准获客
   - **代表产品**：Jewel Blast 曾入选中国出海游戏 Top 10
   - **荣誉**：2024 年获 AppsFlyer MAMA 营销增长奖
   - **月活**：自有游戏矩阵 1000 万+ MAU

### 3.2 商业模式解构

| 维度 | 小迈科技 | 至真科技 |
|------|---------|---------|
| **主要收入模式** | 广告变现（IAA） | 广告技术服务费 + SaaS 订阅 + 游戏发行分成 |
| **收入来源数量** | 1 条（广告） | 3 条（广告 + SaaS + 游戏） |
| **客户类型** | 主要 ToC（终端用户） | ToB（广告主、电商商家）+ ToC（游戏用户） |
| **定价模式** | 免费 APP + 广告展示 | CPI/CPA 效果付费 + SaaS 月费 + 游戏 CP 分成 |
| **核心商业逻辑** | "流量工厂"——批量生产 APP 获取用户，通过广告变现 | "出海基础设施"——为企业提供营销、电商、游戏发行三位一体服务 |
| **单位经济模型** | CPI 买量成本 vs eCPM 广告收入 | 广告服务抽佣率 + SaaS ARPU + 游戏分成比例 |

#### 商业模式深度对比

**小迈科技——"流量变现"模式：**

```
研发成本(开发者工资) → 产品上线 → 买量获客(CPI) → 用户使用 → 展示广告(eCPM) → 广告收入
                                                                                    ↓
利润 = Σ(用户LTV × 用户数) - Σ(CPI × 安装数) - 研发成本 - 运营成本
```

- **关键指标**：eCPM（每千次展示收入）、LTV/CAC 比率、DAU 留存率
- **eCPM 参考基准**（2024 Q4，美国市场）：
  - 激励视频广告：iOS $19.63 / Android $16.49
  - 插屏广告：iOS $14.32 / Android $10-12
  - 横幅广告：$2-5
- **利润率估算**：典型 IAA 应用毛利率 50-70%（扣除买量成本后净利率 10-30%）
- **风险**：Apple ATT 导致 iOS 端精准投放效率下降，ATT opt-in 率已从 2021 年的 26% 降至 2024 年不足 14%（[adjoe](https://adjoe.io/glossary/idfa-identifier-for-advertisers/)）

**至真科技——"平台服务"模式：**

```
引擎1: 广告主预算 → NetMarvel 投放 → 按效果(CPI/CPA)收费 → 抽佣/差价
引擎2: 电商商家 → Shoplus 建站 → 月费 + 0.5%交易佣金
引擎3: 游戏CP → GameGoing 发行 → 流水分成(通常 30-50%)
```

- **广告引擎盈利模型**：DSP 端向广告主收费 - SSP 端向开发者分成 = 平台利润（行业 Margin 通常 15-30%）
- **SaaS 引擎盈利模型**：(月费 + GMV × 0.5%) × 商家数，低客单但规模化后边际成本极低
- **游戏引擎盈利模型**：代发行抽佣（通常 30-50% 流水）+ 自研游戏广告收入
- **优势**：三引擎之间可交叉导流——NetMarvel 为 Shoplus 商家提供广告投放服务，GameGoing 为 NetMarvel 贡献优质流量

### 3.3 技术能力对比

| 维度 | 小迈科技 | 至真科技 |
|------|---------|---------|
| **核心技术资产** | 聚量、洪流、天眼、天启四大自研平台 | SSP/DSP/ADX 程序化广告全栈 + AI 大数据 |
| **数据处理规模** | 日数据量 100 亿+ | 月广告请求 3000 亿+（日均约 100 亿） |
| **AI/算法能力** | 广告投放优化、用户画像 | 机器学习竞价、预算分配、受众定向、创意优化 |
| **知识产权** | 161 软著 + 3 专利 + 31 商标 | 未公开具体数量 |
| **云架构** | 阿里云 SAE 微服务架构（[阿里云案例](https://help.aliyun.com/zh/sae/product-overview/internet-technology-sae-helps-xmiles-improve-microservices-r-and-d-efficiency)） | 未公开 |
| **技术深度** | 应用级（APP 开发 + 广告 SDK） | 基础设施级（广告交易所 + SaaS 平台 + 数据平台） |

> **分析：** 两家公司的日数据处理量在同一量级（约 100 亿级），但技术深度差异显著。小迈科技的技术主要服务于自有产品的运营优化，是"内向型"技术能力。至真科技的技术需要支撑外部客户的多样化需求（广告投放、电商建站、游戏发行），是"外向型"技术能力，复杂度和可扩展性要求更高。

### 3.4 供应链与运营模式

| 维度 | 小迈科技 | 至真科技 |
|------|---------|---------|

  ```
  NetMarvel 流量 → 服务广告主 → 数据积累 → 投放效率提升

  GameGoing 游戏 → 自有流量增长 → 反哺 NetMarvel 供给
  ```

### 9.2 战略方向差异

**小迈科技 — 纵向深耕：**
- **核心策略**：持续打磨"批量化产品开发→广告变现"的效率
- **产品策略**：从纯超休闲向混合休闲（IAA+IAP）升级，提升单用户 LTV
- **技术策略**：完善聚量/洪流/天眼/天启四大系统的协同效率
- **潜在方向**：可能开放开发者孵化平台，从"自研为主"转向"平台+自研"
- **上市计划**：2022 年入选拟上市百强，但后续进展不明——可能受合规问题和市场环境影响

**至真科技 — 横向扩展：**
- **广告引擎**：NetMarvel 持续提升 AI 投放能力，扩大广告主覆盖
- **电商引擎**：Shoplus 深度绑定 TikTok 电商生态，用低价策略抢占中小商家
- **游戏引擎**：GameGoing 从纯发行拓展到研发+发行+代投全链路
- **新赛道**：布局**短剧出海**营销服务——2025 H1 海外短剧市场收入 $10.88 亿（+249%），至真以 NetMarvel 的投放能力切入
- **生态协同**：三大业务线的数据和流量交叉，构建出海企业的一站式服务平台

## 十二、结论与展望

### 总体评价

经过十余年发展，两家同年同城成立的出海公司已走出截然不同的路径：

- **小迈科技**是一家"产品驱动型"公司，核心能力在于**批量化 APP 开发 + 广告变现**，走"自己赚钱"路线。其优势在于团队执行力强、产品迭代快，但面临**收入模式极度单一、市场趋于饱和、合规风险暴露**的三重挑战。在 Apple ATT 持续冲击和超休闲游戏下载量见顶的大背景下，小迈科技若不开辟第二增长曲线，将面临增长瓶颈。

- **至真科技**是一家"平台驱动型"公司，核心能力在于**广告技术平台 + 跨境电商 SaaS + 游戏发行**，走"帮别人赚钱"路线。其优势在于业务多元化、所处赛道高速增长、生态协同效应强。但在全球广告市场高度集中（AppLovin 市值超千亿美元）的格局下，至真科技需要在**品牌建设和技术深度**上持续投入，才能在巨头夹缝中保持增长。

### 关键亮点

1. **至真科技的三引擎模式**（NetMarvel + Shoplus + GameGoing）在中国出海赛道中**较为独特**——同时具备广告技术、电商 SaaS 和游戏发行能力的公司非常罕见，这种结构为交叉销售和数据协同创造了显著优势
2. **至真科技五年 30 倍营收增速**（截至 2019）+ 连续获评"未来独角兽"，验证了其商业模式的可扩展性
3. **小迈科技创始人的久邦数码经验**为团队提供了扎实的移动产品方法论，161 项软著积累体现了较强的技术沉淀

### 需关注的风险

1. **小迈科技的合规风险尚未完全化解**——2021 年工信部通报后，其上市进程疑似受阻，未来出海合规（GDPR/CCPA）压力会更大
2. **两家公司的财务数据均高度不透明**——在没有经审计财务数据的情况下，对其实际经营状况的判断存在较大不确定性
3. **Apple ATT 对 IAA 模式的结构性冲击**——iOS opt-in 率降至 14% 以下，约 10% 移动广告收入已转向上下文定向，这对以广告变现为核心的两家公司都有影响，但对小迈科技的冲击更为直接

### 未来 1-3 年展望

| 维度 | 小迈科技 | 至真科技 |
|------|---------|---------|
| 增长预期 | 低——维持或微增（0-20%/年），除非开辟新业务线 | 中高——20-40%/年（受益于程序化广告和电商 SaaS 增长） |
| 关键看点 | 是否向混合休闲转型？是否重启上市？是否开放开发者平台？ | Shoplus 能否在 Shopify/店匠的竞争中突围？短剧出海能否成为新增长点？ |
| 潜在风险事件 | 更多合规通报 / 核心 APP 被下架 / 上市计划搁浅 | AppLovin 等巨头进一步整合挤压空间 / 地缘政治影响出海业务 |
| 建议关注信号 | 新产品线或业务模式公告 / 新一轮融资 / 上市申报 | C 轮或后续融资 / Shoplus GMV 增速 / 短剧业务进展 |

> **从海外业务发展潜力看，至真科技的赛道更宽、天花板更高、抗风险能力更强。** 其"广告+电商+游戏"三引擎模式在中国企业出海大潮中有更大的服务空间。小迈科技若不突破单一 IAA 模式的天花板，将面临增长停滞的风险。
