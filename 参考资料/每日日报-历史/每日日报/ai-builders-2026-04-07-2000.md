**Peter Yang（Roblox 产品）**

Peter Yang 与 a16z GP @illscience 进行了一场关于 AI agent 时代的深度对话，核心观点：

Coding 会吞噬所有知识型工作——写文档、做 PPT、拉数据，先用 AI coding agent 完成 80% 的初始工作，再人工打磨最后 20%，已经不需要从零开始了。2-3 人团队 + 一群 agent 会取代臃肿的组织，曾经需要 3 小时 OKR 会议来对齐的大公司，正在被刻意保持小规模的新一代创始人绕过。而更有意思的是预测：随着个人 AI agent 越来越懂你，它会开始提醒你真正重要的事——"你的孩子才 7 岁和 4 岁，他们长大得很快，多陪陪他们。" Agent 不会让野心触顶，只会让经济的形态改变。

**Aaron Levie（Box CEO）**

Levie 对 AI agent 时代的工作模式给出了一个反直觉的判断：Agent 把工作往上推了一层抽象，但现在的工作变成了——搞清楚让 agent 干什么、给出正确的指令、提供足够的上下文、在它跑偏时介入、审查最终结果、把输出整合进更大的流程。这其中任何一个环节出问题，产出就是废品。任务周期越长，规划和 review 阶段占的比重反而越大。"你现在成了编辑、成了 manager、成了 producer。品味依然重要，对所从事领域的理解依然重要，技能依然重要。"
