# OpenAI 扩张 Daybreak：把「找漏洞 + 修漏洞」做成政府背书的护城河

**一句话定性：** OpenAI 在 6/22 把 Daybreak 升级为完整的「攻防一体」平台——公开发布能找漏洞、写 exploit 的 GPT-5.5-Cyber，把安全扫描嵌进 Codex 开发回路，并发起机器速度修补开源的 "Patch the Planet"；而**就在 11 天前，Anthropic 的 Fable 5 / Mythos 5 因为同一类「读代码找漏洞」的双用途能力被美国政府出口管制下线、至今未复线**。同一种能力，一个成了国家安全负债、一个成了政府背书的护城河——差别只在「许可层」。

## 为什么值得深读 / 背景

过去三周，「前沿模型的攻击性网络安全能力」从研究议题变成了**行业格局的主轴**：

- 6/12，美国商务部以「越狱风险」为由出口管制 Anthropic 的 Fable 5 / Mythos 5——而所谓「越狱」的实质，是让模型**读代码库、找软件漏洞**这一标准防御能力（见账本 6/20、6/21 两篇）。截至今日（6/23）已第 11 天仍未复线。
- 6/18，Google DeepMind 发布 AI Control Roadmap，首次系统承认「对齐不可靠、要靠系统级遏制兜底」（账本 6/22）。
- 6/18，微软用 AutoJack 三漏洞链证明「localhost 不再是信任边界」（账本 6/20）。

OpenAI 这次扩张是这条主线上**最具攻击性、最商业化**的一手动作：它不只是研究承诺，而是把「找漏洞→验证→写补丁」整条链路产品化、嵌进开发者每天用的 Codex，并配上一套**只给「可信防御者」的分级许可**。它同时回答了一个所有做 agent 的团队都绕不开的问题：当模型既能找漏洞又能写 exploit，**你靠什么决定谁能用、怎么不被监管一刀切**？

## 核心事实（硬数据分层）

**1）三件套（6/22 同时发布）**

- **完整版 GPT-5.5-Cyber**：面向「经验证的防御者」、用于授权环境下的代码库分析、可达性检查、漏洞验证与补丁开发。OpenAI 设三档变体：
  - `GPT-5.5`（标准安全护栏，通用）
  - `GPT-5.5 Trusted Access for Cyber`（经验证的防御性工作）
  - `GPT-5.5-Cyber`（**宽松护栏**，专供红队 / 渗透测试 / 受控验证）
- **Codex Security 插件**：把扫描嵌进 Codex 开发回路——commit 审查、严重度报告、修复建议、**攻击路径追踪**、威胁建模、补丁生成。3 月预览以来：**扫描 3000 万+ commits、30,000+ 代码库；70,000+ 发现被审查者标记为已修复、500,000+ 被自动检测为已修复**。
- **Patch the Planet**：与 **Trail of Bits、HackerOne** 及安全研究者共建，**30+ 开源项目**承诺参与，含 **cURL、Go、Python、Sigstore、pyca/cryptography、NATS Server、aiohttp、freenginx、Python.org**；为参与项目提供 ChatGPT Pro 访问、Codex Security 条件访问与 API 额度。
- **Cyber Partner Program**：把 GPT-5.5 Trusted Access for Cyber 嵌入既有安全产品；Trusted Access 已对接 **Akamai、Cisco、Cloudflare、CrowdStrike、Fortinet、Oracle、Palo Alto Networks、Zscaler**。

**2）能力跃迁（GPT-5.5-Cyber vs GPT-5.5 基线）**

| 基准 | GPT-5.5-Cyber | GPT-5.5 |
|---|---|---|
| CyberGym | **85.6%** | 81.8% |
| ExploitGym | **39.5%** | 25.95% |
| SEC-bench Pro | **69.8%** | 63.1% |

> 注意 **ExploitGym 从 25.95% 跃到 39.5%**——这衡量的是**写出可用 exploit** 的能力，提升幅度（+52%）远大于纯发现类基准。攻击性能力是这次最实打实的增量。

**3）关键原话**

- OpenAI：「We want to help all companies be secure, working with the USG and the security ecosystem.」（强调与美国政府 USG 协作——许可层的政治锚点）

## 机制拆解：这套设计真正在解决什么

表面是「安全产品」，底层是 **OpenAI 对双用途风险给出的治理范式**，三个杠杆：

1. **能力分级 × 身份分级**：同一基座模型切成三档护栏，把「谁是可信防御者」做成准入开关，而不是把能力本身砍掉。这正是 Anthropic 在 Fable 5 上**想做却来不及做**的工程切分——Anthropic 因「无法按国籍工程切分」只能对全球全量停服。
2. **左移进开发回路**：Codex Security 让「写 bug 的工具」和「找 bug 的工具」合一——FourWeekMBA 称之为**平台锁定**：「write the bug and find the bug in the same tool」。安全扫描从事后审计变成 agent 生成代码时的原生步骤。
3. **政府关系即护城河**：反复绑定 USG。批评者直言，两家实验室都得出同一结论——**网络安全是前沿能力「最即时有用、且政治上最可辩护」的场景**。Glasswing 在 Fable 5 被下线时安然无恙，等于证明了「防御性安全 = 受保护领域」。

## 正反观点

**支持方（OpenAI / 防御者视角）：**
- 攻击者已在用 AI 加速找漏洞，防御方必须对等武装；分级许可 + 与安全生态 / USG 协作是「负责任地释放双用途能力」的可行解。
- 硬数据支撑：30M commits、57 万处修复，已是规模化的真实防御产出，不是 demo。

**质疑方 / 风险：**
- **「双用途」并未被解决，只是被许可层包装**。Hacker News 指出：AI 把漏洞发现提速到维护者跟不上——3 月 HackerOne 已暂停其互联网漏洞赏金计划，出现「triage fatigue」与幻觉报告泛滥。研究者 Himanshu Anand 一针见血：「当 10 个互不相关的研究者六周内找到同一个 bug，而 AI 能在 30 分钟内把补丁 diff 变成可用 exploit，90 天披露窗口到底在保护什么？」
- **监管捕获之嫌**：批评者认为两家在争「钱买不到的东西——监管保护」，靠政治上可辩护的安全工作换准入，而非在开放市场拼能力。
- **攻击性能力被公开放大**：ExploitGym +52% 的同一能力，落到「宽松护栏」变体里，准入边界是否真的守得住，存疑。

## 深度解读：对行业 + 对「做多 agent 系统的我们」的具体含义

**对行业——能力在商品化，「许可层」成了新的竞争维度。** Fable 5 与 GPT-5.5-Cyber 是同一种双用途能力的两种命运：一个被监管一刀切下线（因为没有许可分层、无法按身份切分），一个被政府背书成护城河（因为有 Trusted Access 分级 + USG 关系）。**结论：前沿能力本身正在拉平，真正的差异化与监管生存能力，落在「谁能用 / 怎么审计 / 与监管什么关系」的治理包装上。** 这对所有要触碰敏感能力的实验室都是模板。

**对我们（做 agent 系统）——四条可执行判断：**

1. **把「安全扫描」当 agent 回路的原生步骤，而非事后挂件。** Codex Security 已把 AppSec 左移进生成回路；我们对 agent 生成的代码也应内置「生成即扫描 + 攻击路径追踪」，否则会被「写 bug、找 bug 同一工具」的对手在 DX 上拉开差距。

2. **任何能读你代码库的 agent，都要按「潜在漏洞发现者」对待。** ExploitGym 39.5% 意味着「读代码→写 exploit」已是现实能力。给读码 agent 上**scoped 凭据 + 全量审计日志 + 危险原语 allowlist**（与 6/22 DeepMind Control Roadmap 的 R1–R3 动作分级一致）。

3. **把「分级许可 / 可信访问」做进产品，从第一天起。** 监管生存与商业护城河现在是同一件事：如果你的 agent 触碰安全敏感能力，「谁经过验证、审计链是否完整、与监管什么关系」不是成本，是**护城河 + 不被一刀切的保险**。Anthropic 的教训是反例——能力先行、切分滞后。

4. **供应链风险窗口在压缩，模型供给要冗余。** Patch the Planet 让你的开源依赖可能被机器速度修补，但「30 分钟出 exploit vs 维护者修补能力」的不对称意味着**依赖风险窗口在收窄**——pin + 监控 + 自动补丁流水线要补上。同时，顶级能力的可得性如今取决于「治理身份」而非「预算」（Anthropic 被出口管制、OpenAI-Cyber 限可信防御者）——**多模型 / 跨阵营冗余热切换**继续是架构刚需。

## 接下来盯什么（信号）

- **GPT-5.6 是否本周发布（窗口 6/22–28，预测市场~83%）**，以及是否带 cyber 变体——若带，攻防一体叙事再升级，立即头号候选。
- **Fable 5 / Mythos 5 复线条件**：是否给「网络安全能力」单独豁免 / 划成受保护领域（呼应 Glasswing 模板）。
- **Daybreak Trusted Access 是否获得正式 USG 认定 / 出口管制豁免**——「政府背书=护城河」能否被坐实。
- **Patch the Planet 的真实产出**：规模化修补的实际 CVE 数、HackerOne triage-fatigue 是否缓解。
- **Google 的对位动作**：DeepMind 已有 AI Control Roadmap + Gemini 安全线，看其是否商品化为同类平台。
