# OpenAI 2025 审计财务泄露：385 亿亏损背后，AI 补贴时代正在终结——而买单的将是 agent

**一句话定性：** OpenAI 上市前夕被泄出的审计数据，第一次让整个行业看清"前沿 AI 的真实成本结构"；而当 agent 把 token 消耗推向递归式爆炸时，过去两年靠"包月订阅 + 资本补贴"撑起来的低价幻觉，正在结构性瓦解——这恰恰是做多 agent 系统的团队最该读懂的一份财报。

**来源：** Fortune（经 Financial Times 独立核实）、Tech Times、Business Model Analyst、Gizmodo、HBS *Future-Proof* 6 月备忘录
**🔗 原始链接：** https://fortune.com/2026/06/16/openai-financials-leaked-losses-revenue-profit/
**日期：** 2026-06-16（IPO 前的审计文件泄露，本周持续发酵）

## 为什么值得深读 / 背景

OpenAI 已于 6 月 1 日向 SEC 秘密递交 S-1，目标年内上市。就在招股书仍处保密期时，博主 Ed Zitron 拿到一份**审计版** 2025 财务文件，并由 *Financial Times* 独立核实。这是迄今为止外界第一次看到 OpenAI 经过审计、而非 Altman 口头"营收远超 130 亿"式表述的真实账本。

它的分量不在"AI 公司烧钱"这个老结论，而在于：
1. **时点**——上市前夕，这套数字会直接定价市场对整个前沿 AI 板块的预期；
2. **结构**——它第一次把"模型研发、算力采购、销售获客"的钱怎么花、花给谁（Azure）摊开；
3. **传导**——它和本月另外两条线（GitHub Copilot 6/1 转用量计费、Anthropic Managed Agents 的 session-hour 计价）拼在一起，指向同一个判断：**包月订阅扛不住 agentic 负载，补贴时代要结束了。**

对"做多 agent 系统"的我们来说，这不是一条财经八卦，而是**未来 12 个月 API 价格曲线和 agent 单位成本的先行指标**。

## 核心事实（硬数据）

OpenAI 2025（对比 2024）经审计数据：

| 指标 | 2025 | 2024 | 同比 |
|---|---|---|---|
| 营收 | **$13.07B** | $3.7B | **+253%** |
| 营收成本(COGS) | $7.5B | $2.65B | +183% |
| 研发(R&D) | **$19.18B** | $7.81B | +146% |
| 销售与市场(S&M) | $5.73B | $1.11B | **+416%** |
| 管理(G&A) | $1.57B | $0.91B | +73% |
| 总成本 | **$34B** | $12.48B | +172% |
| **经营亏损** | **$20.92B** | $8.78B | +138% |
| **净亏损** | **$38.53B** | — | — |

关键拆解（决定你怎么读这份报表）：
- 净亏 385 亿里，**约 415 亿是非现金计提**，源于"非营利→PBC（公益公司）"架构转换的一次性会计处理。
- 因此**经营亏损 209 亿才是更有意义的口径**；据测算**真实现金消耗约 80 亿美元**。
- **对 Azure 的依赖约 170 亿**——单一云供应商既是命脉也是议价软肋。
- 效率确实在改善：每 1 美元营收对应的支出从 2024 的 **$2.37 降到 2025 的 $1.60**。
- 资本垫底：3 月完成 1220 亿融资、投后估值约 8520 亿；Altman 目前**持股为零**（PBC 转换后据传谈判约 7%）。

## 机制拆解：为什么"agent"会把这张表推向更危险的方向

这份报表是"聊天时代"的成本结构。真正的拐点在于负载形态正在从**一次问答**变成**自主多步 agent**，而 token 消耗是**递归、非线性**的：

- 斯坦福数字经济实验室：**基础 agent 每个任务 5,000–15,000 token**；**复杂工作流单个目标常达 20 万–100 万+ token**。
- 现实案例：一个未加监控的 agent 因递归循环**一夜跑出 437 美元账单**。
- 前沿模型当前定价约 **$5 / 百万输入、$25 / 百万输出 token**——单看很便宜，乘上递归负载就不便宜。

把它和本月的定价动作叠起来看，方向一致：
- **GitHub Copilot**：6/1 起全线转**用量计费**（此前社区抱怨单次 agentic session 烧掉 30–40 美元、相当于 Pro 月度额度的 3–4 倍）。
- **Anthropic Managed Agents**（4/8 上线）：标准 per-token 推理费 **+ agent 运行时 $0.08 / session-hour** 的混合计价。
- **OpenAI**：ChatGPT Plus 将 GPT-5 Thinking 限到**每周 3,000 条**，超出降级到更便宜的模型；Workspace Agents 走信用点制。

机制很清楚：**flat-rate 订阅在"人类按分钟读"时还能用平均数兜底；一旦 agent 7×24 自驱、token 按指数膨胀，平均数失效，供应商必须把成本透传给用户。** OpenAI 这张 209 亿经营亏损的表，就是"还在补贴"的量化证据——而补贴不可能在上市后无限继续。

## 正反观点

**支持/多头主张：**
- 营收一年翻三倍多（37 亿→131 亿），**单位效率在改善**（$2.37→$1.60）；
- 385 亿净亏的大头是 PBC 转换的**一次性非现金计提**，现金消耗约 80 亿，对一家投后 8520 亿、手握 1220 亿融资的公司可控；
- 这是"先占规模、再收价"的经典打法——一旦上市拿到公开市场弹药、再叠加提价/企业版/AI 云，转正路径成立。

**质疑/风险方：**
- 经营亏损一年扩大约 8 倍口径、**烧钱速度跑赢营收**；
- 对单一云（Azure 170 亿）的**结构性依赖**既是成本也是议价弱点；
- 最关键：**真实成本一旦在上市后被市场强制透明，"低价 API/包月"的补贴就难以为继**——而 agentic 负载恰恰在此刻把单位成本推高。两股力量对撞，受冲击的不是 OpenAI 的估值，而是**所有建立在"token 很便宜"假设上的产品**。

## 深度解读：对行业 + 对"做多 agent 系统的我们"的具体含义

**对行业：** 这份泄露把"前沿模型公司到底是不是好生意"从信仰问题变成了可计算问题。上市定价会给整个板块定锚；而它和 Copilot/Anthropic 的计价转向共同确认了一个拐点——**行业正从"用补贴抢用户"切换到"用真实单位成本定价 agent"**。换句话说，2024–2025 的低价是融资补贴出来的，不是成本曲线降出来的。

**对我们（落到可执行判断）：**
1. **把 token 成本当成一等架构约束，而不是事后账单。** 复杂工作流单目标 20 万–100 万 token 是常态，现在按补贴价算的毛利，在补贴退潮后会被重算。每一个 agent 设计评审都要带"递归 token 上界"这一项。
2. **现在就给 agent 装"熔断"。** 那个 437 美元一夜账单是递归循环的典型。硬性上限：单 run token/调用次数 cap、循环检测、超时即停、按 run 计费可视化——别等供应商替你"省钱降级"。
3. **为"计价模式迁移"做预案。** Copilot 已从包月转用量计费、Anthropic 收 session-hour、ChatGPT 加周配额——**假设我们依赖的任何 flat-rate 额度都会在 6–12 个月内被用量计费或限额取代**，提前把成本模型从"订阅位"改成"按 token/按 session-hour"。
4. **用分层路由对冲提价。** 既然前沿定价（$5/$25 每百万）会随补贴退潮上行，把"贵模型只做难步骤、便宜模型/本地模型做大批量"做成默认架构（呼应近期 DiffusionGemma、本地开源模型那条线），是直接的成本对冲。
5. **别把单一供应商的低价当永久。** OpenAI 对 Azure 170 亿的依赖提醒我们：上游议价结构会传导到我们的成本。多供应商可切换 + 把 prompt/工具层与具体模型解耦，是抗提价的工程保险。

一句话：**这份报表不是 OpenAI 的麻烦，是 agent 团队的预警——补贴在退潮，递归 token 在涨潮，谁的 agent 没装成本护栏，谁先被新计价模式打穿毛利。**

## 接下来盯什么（信号）

- **OpenAI 公开版 S-1**：保密期结束后，看官方口径如何呈现现金消耗 vs 非现金计提、以及对 Azure 依赖的风险表述。
- **前沿 API 提价/限额动作**：OpenAI / Anthropic / Google 任一家上调 per-token 价或收紧包月额度，即为补贴退潮坐实。
- **agent 计价标准化**：session-hour、按 run、按工具调用等新计价单位是否扩散成行业默认。
- **OpenAI 毛利与"AI 云"营收**：Altman 的 1000 亿（2027）叙事能否兑现，关键看企业版与 API 云收入占比。
- **Anthropic 对照组**：其 IPO（估值约 9650 亿）招股材料若披露成本结构，可与 OpenAI 这套数据做行业级横评。
