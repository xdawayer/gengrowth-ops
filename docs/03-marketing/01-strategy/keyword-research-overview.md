# 增长营销关键词与转化框架（路由总览）

> **本文档定位**：跨获客方式（SEO / 社媒 / 买量 / 社区 / 合作）+ 跨业务类型 + 跨站点阶段的关键词研究与转化承接的**入口与路由**。
>
> **当前状态**：v0.5 框架草稿。仅搭建维度结构与文档地图，各维度的具体内容由后续专项 SOP 实现。
>
> **使用方式**：先按"决策维度（4 维输入）"定位你的场景，再按"执行链路（4 步骤）"逐步推进；不在已有 SOP 覆盖范围 → 找最接近的 SOP，对照决策维度差异做调整。
>
> **配套**：本文档对应的"GenGrowth 当下落地决策"见 `scenario-priority-matrix.md`。

---

## 一、为什么需要这份框架

单一关键词 SOP 无法覆盖所有营销场景。不同业务类型（B2B SaaS / 电商 / 媒体 / 本地）、不同站点阶段（新站 / 老站 / 头部站）、不同获客方式（SEO / 社媒 / 买量 / 社区 / 合作）、不同目标地区（英文谷歌 / 中文百度），需要的关键词逻辑、内容形态、转化路径完全不同。

**本框架的目标**：建立一张全景图，把"决策维度（输入）"和"执行链路（输出）"显性化，避免在错误的场景套用错误的 SOP，也避免做了关键词没承接、做了内容没转化的断点。

## 三、决策维度（4 维输入）

> 当前为占位说明，待后续专项 SOP 填充实质内容。

### 维度 1：业务类型

不同业务类型决定关键词的搜索意图分布、内容形态偏好、转化路径长度。**变现模式**也影响该业务与 GenGrowth 当前服务模式（订阅付费业务关键词增长 SOP）的契合度。

> 业务类型按 GenGrowth 当前能力契合度 + 商业模式契合度排序（高 → 低）。

| 业务类型             | 变现模式        | 关键词重心                   | 对应 SOP                                                   |
| ---------------- | ----------- | ----------------------- | -------------------------------------------------------- |
| B2C 工具 / 早期 SaaS | 订阅付费        | 信息词 + 工具型查询             | `../03-seo/keyword-research-sop.md`（已有，当前主用）             |
| 电商               | 直接交易（持续付费）  | Transactional 意图 + 类目长尾 | `keyword-research-sop-ecommerce.md`（待建，有合作资源）            |
| C 端小游戏 / 休闲游戏    | 广告变现 + IAP  | ASO 关键词 + 付费投放素材关键词     | `keyword-research-sop-casual-game.md`（待建，团队基因强但客户付费意愿中等） |
| B2B SaaS         | 订阅付费        | Commercial 意图 + 长尾决策词   | `keyword-research-sop-b2b-saas.md`（待建，客户基础弱）             |
| 媒体 / 内容站         | 广告变现        | 信息词 + 时效热点              | `keyword-research-sop-media.md`（待建）                      |
| 本地服务             | 服务付费        | 地理修饰词 + 即时意图            | `keyword-research-sop-local.md`（暂缓，缺合作资源）                |
| YMYL（医/金/法）      | 服务付费 / 内容订阅 | 权威解释词 + E-E-A-T         | `keyword-research-sop-ymyl.md`（暂缓，缺合作资源）                 |

> **GenGrowth 服务模式契合度过滤**：当前主力服务对象是订阅付费业务的关键词增长 SOP 咨询。广告变现业务（休闲游戏 / 媒体）的客户核心 KPI 是 ARPDAU / RPM / 留存，与 SEO 增长产出物匹配度中等；客户对外部咨询的付费意愿也中等。详见 `scenario-priority-matrix.md` 评分方法论 2.4 节。

### 维度 2：站点阶段

不同 DR 阶段决定 KD 阈值、产能分配、是否启动刷新与修剪。

| 阶段   | DR 区间 | 重心                                    |
| ---- | ----- | ------------------------------------- |
| 新站起步 | 0–20  | 极小词 + 长尾矩阵 70%+ + Featured Snippet 抢位 |
| 增长期  | 20–40 | 快速胜利 60% + 长尾 40%（当前主 SOP）            |
| 成熟期  | 40–60 | 新内容 30% + 刷新 70% + 上位词挖掘              |
| 头部期  | 60+   | Topic Authority + 内链权重分配 + 修剪         |

> 待建：`keyword-research-sop-mature-site.md`（增长期+成熟期）、`keyword-research-sop-head-site.md`（头部期）

### 维度 3：目标地区

不同地区使用不同的搜索引擎、工具链、内容生态。

| 地区                     | 主搜索引擎         | 关键词工具                         | 内容生态                        | 优先级                                   |
| ---------------------- | ------------- | ----------------------------- | --------------------------- | ------------------------------------- |
| 英文谷歌（美国为主，含英 / 加 / 澳新） | Google        | Ahrefs / SEMrush / DataForSEO | YouTube / Reddit / Quora    | **Tier 1**                            |
| 巴西葡语 / 拉美西语            | Google        | Ahrefs（葡 / 西语支持）              | YouTube / Discord / Reddit  | **Tier 1**                            |
| 东南亚 / 其他多语言            | Google + 本地引擎 | 各地区工具组合                       | TikTok / Telegram / Discord | Tier 2                                |
| 中文百度系                  | 百度 / 神马       | 5118 / 站长之家 / 爱站              | 知乎 / 小红书 / 视频号 / 抖音         | **不规划**（团队不擅长百度系，与 Google/Bing 市场差异大） |

> **GenGrowth 团队基因锚点**：以美国 + 巴西为主的休闲游戏与工具研发发行经验，与 Google 谷歌生态强相关。中文百度系短期不拓展。
>
> 待建：`keyword-research-sop-portuguese.md`（巴西葡语 + 拉美西语）、`keyword-research-sop-multi-region.md`（东南亚等）

### 维度 4：获客方式

获客方式分四大类，每类的关键词逻辑（或"非关键词逻辑"）不同：

- **内容驱动型**（SEO / 社媒 / 社区）：共用大部分关键词方法论，是本框架的**当前核心层**——主要服务订阅付费业务（B2C 工具 / B2B SaaS）
- **付费驱动型**（SEM / 信息流 / 应用商店）：是 GenGrowth **下一阶段战略重点**——订阅付费业务在内容驱动有起步流量后的扩量放大器；游戏发行场景下付费投放是主战场
- **关系驱动型**（合作分销 / KOL）：作为协同补充，不独立成主战场
- **直接交易型**（电商搜索 / 类目流量）：电商业务的主战场，独立于内容/付费驱动

> **关键映射**：B2C 工具 / B2B SaaS → 内容驱动 + 付费驱动双轮；电商 → 直接交易 + 付费驱动 + 内容驱动；游戏 → 付费驱动为主（不靠内容获客）；媒体 → 内容驱动（但变现弱）。

#### 4.1 内容驱动型（核心层）

| 子类 | 关键词角色 | 对应 SOP |
|---|---|---|
| SEO（自然搜索）| 主战场，搜索意图与 KD 优先 | 见维度 1 各业务类型 SOP |
| 社媒营销（Twitter / LinkedIn / 小红书 / 视频号）| 话题词 + 标签词，意图弱、共鸣强 | `../02-social-media/social-keyword-sop.md`（后续建）|
| 社区运营（Reddit / Quora / IH / 知乎）| 真实用语 + 趋势话题 | 见 SEO SOP 来源 4 |

#### 4.2 付费驱动型（下一阶段战略重点）

> 当前 GenGrowth 团队付费基因正在补建。在订阅付费业务模式下，付费驱动是规模化的关键放大器；游戏发行场景下付费投放是标配。

| 子类                            | 关键词角色              | 对应 SOP                                       |
| ----------------------------- | ------------------ | -------------------------------------------- |
| SEM（Google Ads）               | 商业意图词 + 出价 + 落地页匹配 | `../paid-acquisition-keyword-sop.md`（后续建，P0） |
| 信息流投放（Meta / TikTok）          | 兴趣标签 + 创意素材，关键词角色弱 | 同上                                           |
| 应用商店投放（ASA / Google Play UAC） | 应用商店关键词与竞价         | 同上（游戏与工具应用必备）                                |

#### 4.3 关系驱动型

| 子类 | 关键词角色 | 备注 |
|---|---|---|
| 合作分销（联盟 / 渠道）| 合作方关键词协同 | 关键词非主导 |
| KOL 投放 / 红人合作 | 话题协同，非搜索关键词 | 关键词非主导 |

#### 4.4 直接交易型

| 子类 | 关键词角色 | 备注 |
|---|---|---|
| 电商搜索（Amazon / Shopify / MercadoLibre）| 商品关键词 + 长尾品类 | 在电商专项 SOP 中处理 |
| 类目流量 | 类目长尾词 | 同上 |

## 四、执行链路（4 步骤输出）

> 决策维度（4 维）确定后，按以下 4 步骤推进执行。每一步是上一步的承接；漏掉任何一步都会让上游工作失效。

### 步骤 1：关键词筛选与优先级

沿用「六源挖掘 → 四桶分级」方法论。详见 `../03-seo/keyword-research-sop.md`。

核心闸门：
- **DR 差距闸门**：Top10 平均 DR 与自有站差距 ≤ 30
- **KD 与意图分层**：四桶（趋势词 / 快速胜利 / 长尾词 / 战略词）
- **AIO 风险标注**：AI Overview 高风险词改用交互工具 / 对比型切入
- **GEO 引用潜力**（v2.2 起）：词型快筛 + Perplexity 实测验证两步法

> 不同获客方式（维度 4）下，"关键词筛选"的具体方法不同：
> - 内容驱动（4.1）：六源四桶（本框架核心层）
> - 付费驱动（4.2）：商业意图词 + 出价模型 + 落地页匹配
> - 关系 / 直接交易（4.3 / 4.4）：见各自专项 SOP

### 步骤 2：内容承接（页面类型）

| 页面类型           | 适配关键词                      | 承接职能           |
| -------------- | -------------------------- | -------------- |
| 落地页            | Transactional / Commercial | 短路径转化          |
| 工具页            | Informational + 工具型查询      | 即时使用 + 留资      |
| 文章博客           | Informational + 长尾         | SEO 主战场 + 教育引导 |
| 比较页（vs / best） | Commercial                 | 决策辅助 + 引导 CTA  |
| 类目 / 产品页       | Transactional 类目长尾         | 电商主战场          |
| 资源中心 / 集群中枢    | 长尾矩阵                       | 集群权重 + 长尾流量汇聚  |
|                |                            |                |

> 待建：`page-type-design-sop.md`

### 步骤 3：产品承接

关键词流量进入后，产品侧需要承接。常见承接形态：

- **免费试用**（产品试用版本 / 沙箱）
- **免费工具**（SEO 工具站常用，工具即内容、内容即工具）
- **注册转化**（账号体系，用户进入产品池）
- **付费订阅**（直接订阅 / 试用转付费）
- **留资询盘**（B2B SaaS 常用，进入销售跟进）

> 待建：`product-funnel-handoff-sop.md`（与产品方协作设计）

### 步骤 4：转化路径

| 路径类型 | 适用场景 |
|---|---|
| 直接转化 | 强 Transactional 意图 + 短决策周期（电商、低价订阅）|
| 多步漏斗 | B2B 长决策 / 高客单（试用 → demo → 销售跟进）|
| 触发跟进 | 留资 → 邮件序列 / 销售跟进 / 再营销 |
| 留存激活 | 注册后激活、付费后留存（生命周期运营）|

> 待建：`conversion-funnel-sop.md`

## 六、使用流程

1. **定位决策维度**：业务类型 + 站点阶段 + 目标地区 + 获客方式 的组合
2. **找对应 SOP**：在「SOP 文档地图」找到该组合对应的专项 SOP
3. **降级路径**：不在已有 SOP 覆盖范围 → 找最接近的 SOP，对照"决策维度"差异做调整
4. **按执行链路推进**：步骤 1 关键词筛选 → 步骤 2 页面类型 → 步骤 3 产品承接 → 步骤 4 转化路径，避免做了关键词没承接、做了内容没转化

## 七、附录：面向投资人的业务优先级说明

> **使用场景**：本附录用于商业计划书、融资沟通和对外业务叙事。它不是内部 SOP 建设排期的机械复制，而是从投资人视角说明 GenGrowth 为什么优先选择这些业务方向。

GenGrowth 的早期业务优先级不是单点押注，而是围绕三类高确定性场景展开：**工具型 / 内容型数字产品、休闲游戏、电商交易增长**。这三类业务分别对应方法论验证、团队经验优势和商业化变现能力，能够共同支撑 GenGrowth 从增长服务走向产品化增长系统。

### 第一优先级：工具型 / 内容型数字产品增长

GenGrowth 首先聚焦工具型和内容型数字产品，包括 AI 工具、效率工具、垂直资源站、内容产品、轻量订阅产品等。这类产品天然依赖搜索需求、内容矩阵和用户意图承接，适合通过关键词研究、内容布局、免费工具、资源页和转化路径设计获得低成本流量。

这一方向的战略意义在于：它最适合沉淀 GenGrowth 的底层增长方法论。GenGrowth 自身也将作为 customer-zero 和样板产品，先用自己的产品验证关键词增长、内容承接、转化路径和自动化 SOP，再将能力服务于外部同类客户。

### 第二优先级：休闲游戏增长与发行

休闲游戏是 GenGrowth 必须放在前面的业务方向。团队在美国、巴西等市场具备休闲游戏研发、发行和增长经验，也有相应资源基础。该方向的增长核心不是传统 SEO，而是付费投放、素材测试、ASO、留存优化、广告变现和地区扩展。

休闲游戏的价值在于，它能发挥团队已有经验优势，并验证 GenGrowth 在全球化发行和买量增长上的能力。相比单纯内容增长，游戏业务更接近结果导向：获客成本、留存、变现、ROI 都可以被快速量化，因此适合作为早期重点业务线之一。

### 第三优先级：电商交易增长

电商同样应放在前排。电商客户有明确交易目标、持续优化需求和较强预算意识，商业化路径清晰。GenGrowth 可以围绕 Amazon、Shopify、MercadoLibre 等平台和独立站，提供商品关键词、类目流量、内容种草、平台搜索、广告投放和转化优化能力。

电商方向的战略价值在于，它直接连接交易结果，能够帮助 GenGrowth 验证从流量增长到销售转化的完整闭环。尤其在美国和拉美市场，电商增长与团队已有国际化经验、关键词能力和付费投放能力存在明显协同。

### 三条主线的关系

这三条业务不是互相替代，而是共同组成 GenGrowth 的早期业务组合：

- **工具型 / 内容型数字产品**：负责沉淀可复制的增长方法论，并以 GenGrowth 自身作为 customer-zero 验证样板。
- **休闲游戏**：负责发挥团队已有发行经验和全球化增长优势，验证买量、ASO、留存和变现能力。
- **电商交易增长**：负责验证更直接的交易转化和商业化能力，补充平台搜索、广告投放和销售转化场景。

因此，GenGrowth 的业务优先级可以概括为：**以工具型 / 内容型数字产品作为方法论样板，以休闲游戏作为经验优势业务，以电商交易增长作为商业化扩展方向，逐步形成覆盖内容增长、付费增长、发行增长和交易转化的综合增长系统。**
