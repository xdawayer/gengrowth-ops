# 外链SOP：如何找能够发外链网站

发外链，有一个可复制的办法：

1. 在 toolify.ai 和 theresanaiforthat.com 找到跟你的产品类似的产品，尽可能的多找一些。
    
2. 去 Semrush 一个一个查这些产品的外链，到处外链域名列表。
    
3. 把所有的外链域名列表汇总到一起，按照出现次数从高到低排序。 你就得到了一个绝大多数网站都能拿到的外链列表。
    
4. 按照这个列表，一个一个去发外链。
