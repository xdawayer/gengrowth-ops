# 工作台上手指南（彭满）

> 欢迎加入 GenGrowth！这份指南带你把"在电脑上写东西 → 自动同步到团队云端"这条路跑通。
> **你不用懂技术。** 最难的一次性配置会**和 wzb 一起做**，你日常只管写。任何地方卡住，**截图发 wzb**。

---

## 1. 先花 2 分钟搞懂这是怎么回事

我们的工作方式，三样东西配合：

| 名字 | 打个比方 | 你怎么用 |
|---|---|---|
| **GitHub** | 团队放资料的"云端总仓库"，在网上 | 你写的东西自动存到这里，团队能看到 |
| **Obsidian** | 你电脑上写东西的"本子" | 你每天就在这里打字、写文章 |
| **Git** | 连接本子和云端的"自动传送带" | 你不用管它，它在背后自动把内容传上云 |

一句话：**你在 Obsidian 写 → 自动传到 GitHub → 团队就能看到。**

> 💡 以后我们还会让 AI 自动接手帮你干活（你写个需求，它帮你做）。那是后话，现在你只要先学会"写 + 自动同步"这一件事。

---

## 2. 你要装的 2 个软件

去官网下载、选对应你电脑系统（Windows / Mac）的版本，像装普通软件一样装好。装不上就问 wzb。

| 软件 | 干嘛用 | 去哪下 |
|---|---|---|
| **Obsidian** | 你写东西的本子 | obsidian.md |
| **Git** | 那条"传送带"的引擎（装好看不到图标，正常） | git-scm.com |

> 你的 GitHub 账号（`Pmemmm`）已经有了，不用再弄。

---

## 3. 和 wzb 一起做一次首次设置（约 15 分钟，做完一劳永逸）

这一步有点技术，**约个时间和 wzb 一起做**。会完成这几件事：

1. **你生成一把"钥匙"（Token）** —— 照**文末附录**做，把那串字符复制好。这是同步时证明"你是你"用的。
2. **wzb 帮你把仓库装到电脑上** —— 他会用你的 Token 跑一段命令，**只把你自己的文件夹 `inbox-pengman` 装下来**（别人的东西不会下载到你电脑，你也碰不到）。
3. **用 Obsidian 打开它** —— 打开后你**只会看到自己的文件夹**，清清爽爽。
4. **打开自动同步** —— 装 Obsidian 的「Obsidian Git」插件，把**自动备份（推送）和自动拉取都打开**（间隔填 5–10 分钟）。这样你写的东西会自动上云、别人的更新也会自动同步给你。

> 这一步只做一次。做完之后，你每天就只剩"打开 Obsidian 写字"这一件事。

---

## 4. 日常怎么用

1. **只在 `inbox-pengman/` 里写**（你打开后看到的就是它）。里面分了几个子文件夹，各有用途——**先读一遍 `inbox-pengman/README.md`**。
2. **文件起名**用 `年-月-日-主题`，例如 `2026-06-12-星座关键词调研`。别留"未命名"。
3. **写完不用做任何操作**，传送带会自动把它传上云。
4. ✅ **每天结束前，确认一下真的传上去了**（重要）：打开 GitHub 网站 → 进 `gengrowth-ops` 仓库 → `inbox-pengman` 文件夹 → 看你今天写的文件在不在。**在 = 成功**；不在或报错，截图发 wzb。
5. 现阶段**不用管"送审、归档"这些**，先把"写 + 同步"用熟，wzb 通知你再学下一步。

---

## 5. 几条红线

- 🚫 **只动自己的 `inbox-pengman/`**，别碰别人的（正常情况下你也看不到别人的）。
- 🚫 **别乱删、乱拖文件** —— 删除和移动也会自动同步上云，删错了影响团队。不确定就先别动，问 wzb。

## 6. 卡住了怎么办

**任何一步卡住、看不懂、弹了看不懂的窗口 —— 先别乱点，截图发 wzb。** 尤其是同步要你"输密码 / 登录"时（那要填的是 Token，不是 GitHub 密码），让 wzb 帮你弄最快。

## ✅ 怎样算上手成功

> 你在 Obsidian 的 `inbox-pengman/` 里**新建一个文件、写几个字**，过几分钟后，**在 GitHub 网站上能看到这个文件出现了**——恭喜，正式上手！

## 附录：怎么生成 Token（GitHub 专用钥匙）

> 同步时用它代替密码证明身份。和 wzb 一起做首次设置时会用到。

1. 浏览器打开 **github.com**，登录账号 `Pmemmm`。
2. 点右上角**头像** → **Settings（设置）**。
3. 左边菜单拉到**最底下**，点 **Developer settings（开发者设置）**。
4. **Personal access tokens** → **Tokens (classic)** → **Generate new token (classic)**。
5. **Note** 随便起个名（如 `obsidian-同步`）；**Expiration** 选长一点（90 天或 No expiration）。
6. **勾选 `repo`** 这一整项（允许同步文件的权限，必须勾）。
7. 拉到底点 **Generate token**。
8. ⚠️ **页面会显示一长串字符 —— 这就是 Token，马上整段复制存好**（备忘录、密码管理器都行）。**页面一关就再也看不到了**。

> 🔒 Token = 你家钥匙：别发群、别截图给无关的人。万一泄露，回这个页面 **Delete/Revoke** 再生成一个新的。
