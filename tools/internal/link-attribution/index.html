备注 UTM 填写规则
  • utm_source 填具体平台。如社交媒体用 reddit / pinterest / x / tiktok等；KOL 用 kol-{红人}；外链用主域名，比如：abc.com。注意：统一用小写字母，同一个来源不能有两种写法！
  • utm_medium 填渠道类型，只填 social / backlink / kol / newsletter。
  • utm_campaign / cluster_id 填共享表里的 cluster_id，如 CL-AW-05。
  • utm_content / action_id 填这条分发动作/帖子的 action_id，取自 6-ID 体系。
  • 未填写会默认留空：空字段不会写入最终链接。
