# 产品类型分类与类型专项分析维度

先识别产品类型，再应用通用基础模块 + 类型专项模块。

## 产品类型识别

| Type ID | 产品类型 | 示例 | 识别信号 |
|---|---|---|---|
| `SAAS` | SaaS / B2B 软件 | Notion, Slack, Salesforce, Datadog | 订阅定价、B2B 导向、MRR/ARR 指标、企业销售 |
| `CONSUMER` | 消费级应用（社交/内容/工具） | TikTok, Instagram, Spotify, Duolingo | MAU/DAU 导向、广告或免费增值、应用商店分发、病毒增长 |
| `GAME` | 游戏（移动/PC/主机） | Genshin Impact, PUBG Mobile, Roblox, Candy Crush | 游戏机制、应用内购/抽卡、玩家留存、品类分类 |
| `MARKETPLACE` | 平台/市场 | Airbnb, Uber, Etsy, DoorDash | 双边市场、GMV/抽成率、供需动态 |
| `ECOMMERCE` | 电商 / DTC | Shein, Shopify merchants, Warby Parker | 商品销售、AOV、物流、库存、供应链 |
| `FINTECH` | 金融科技/支付 | Stripe, Revolut, Robinhood, Alipay | 金融交易、AUM、监管牌照、合规密集 |
| `AI_DEVTOOLS` | AI / 开发者工具 | OpenAI, Cursor, Vercel, Twilio | API 优先、开发者采用、按 token/用量定价 |
| `HARDWARE` | 硬件 + 软件生态 | Apple, Tesla, Oculus, DJI | 实体产品 + 软件、BOM 成本、ASP、生态锁定 |
| `HEALTH` | 健康/医疗科技 | Calm, MyFitnessPal, Teladoc | 健康数据、临床验证、FDA/CE 合规、订阅 |
| `EDTECH` | 教育科技 | Coursera, Khan Academy, Byju's | 学习成果、完课率、B2C+B2B2C、机构销售 |
| `MEDIA` | 媒体/流媒体/内容 | Netflix, YouTube, Substack | 内容库、订阅用户、内容成本、ARPU |

**混合产品规则**：产品跨多种类型时，以核心商业模式确定 **主类型**，其他适用类型为 **副类型**。
- 主类型 → 决定 4.4 节的主体结构
- 副类型 → 作为 4.4 节内的补充子节
- 示例：Roblox → 主 `GAME`（核心循环、玩家指标、LiveOps），副 `MARKETPLACE`（UGC 经济、创作者收入、抽成率）
- 示例：Duolingo → 主 `EDTECH`（学习指标、完课率、证书），副 `GAME`（核心循环、连续签到机制、游戏化）
- 示例：Spotify → 主 `MEDIA`（内容库、订阅用户、ARPU），副 `MARKETPLACE`（创作者经济、收入分成）

## 类型专项分析维度

### `SAAS` — SaaS / B2B 软件

| 维度 | 关键指标与分析要点 |
|---|---|
| 收入质量 | MRR, ARR, ACV, TCV, 收入增长率 (MoM/YoY) |
| 留存与扩展 | 净收入留存率 (NRR), logo 流失, 收入流失, 扩展收入占比 |
| 单位经济 | CAC, LTV, LTV/CAC, CAC 回收期, 毛利率 |
| 销售模式 | 自上而下（企业）vs 自下而上（PLG），销售周期，AE 产能 |
| 产品驱动增长 | 免费层转化率, 首次价值时间, 激活率, PQL 定义 |
| 企业就绪度 | SOC2/ISO27001, SSO/SAML, RBAC, 审计日志, SLA, 可用性 |
| 集成生态 | API 广度, 应用市场, 关键集成, 合作伙伴生态 |

### `CONSUMER` — 消费级应用

| 维度 | 关键指标与分析要点 |
|---|---|
| 参与度 | MAU, DAU, DAU/MAU 比率, 日均会话数, 单次会话时长 |
| 留存 | D1/D7/D30 留存率, 留存曲线形态, 队列分析 |
| 病毒性 | K-factor, 邀请率, 自然 vs 付费安装比 |
| 变现组合 | 广告 vs 订阅 vs IAP, ARPU, ARPPU, 付费转化率 |
| 应用商店表现 | App Store/Google Play 排名, 评分, 评论情绪, ASO 关键词 |
| 内容与信息流 | 内容创作率 (UGC), 推荐算法质量, 内容审核 |
| 网络效应 | 类型（直接/间接/数据）, 强度, 防御性 |

### `GAME` — 游戏

| 维度 | 关键指标与分析要点 |
|---|---|
| 品类与子品类 | Sensor Tower 分类：休闲、中核、硬核、博彩；子品类（三消、MOBA、RPG 等） |
| 核心循环 | 玩法循环描述、单次会话时长、会话频率 |
| 变现模型 | IAP（抽卡/通行证/外观/体力）、广告（激励/插屏）、买断、订阅 |
| 玩家指标 | DAU, MAU, CCU（并发）, D1/D7/D30/D90 留存, ARPDAU, 付费转化率 |
| LiveOps | 活动节奏、赛季内容、通行证周期、更新频率 |
| 社交与多人 | 社交功能、公会/部落系统、PvP/合作、社区规模 |
| 平台与分发 | 移动/PC/主机、跨平台支持、分发渠道（App Store, Steam, Epic 等） |
| IP 与美术风格 | 原创 vs 授权 IP、美术风格（2D/3D/像素/二次元）、地区审美偏好 |
| 竞争格局 | 品类市场份额、按收入排名前 5 竞品、品类增长趋势 |

### `MARKETPLACE` — 平台/市场

| 维度 | 关键指标与分析要点 |
|---|---|
| 交易指标 | GMV, 净收入, 抽成率/佣金比例, 平均订单金额 |
| 流动性 | 供需比, 匹配时间, 完成率, 地理覆盖 |
| 网络效应 | 跨边网络效应强度, 同边效应, 多归属风险 |
| 供给侧 | 供应商数量, 供应商获取成本, 供应商留存, 供给质量 |
| 需求侧 | 买方获取成本, 买方留存, 复购率 |
| 信任与安全 | 评价/评分系统, 纠纷解决, 反欺诈, 保险/保障 |
| 脱媒风险 | 参与者绕过平台直接交易的可能性 |

### `ECOMMERCE` — 电商 / DTC

| 维度 | 关键指标与分析要点 |
|---|---|
| 销售指标 | GMV, 收入, AOV, 转化率, 购物车放弃率 |
| 客户经济 | CAC, LTV, 复购率, 购买频次 |
| 供应链 | 采购模式（1P/3P/代发）, 履约（自有/3PL）, 配送速度, 退货率 |
| 产品策略 | SKU 数量, 品类广度 vs 深度, 自有品牌占比, 新品节奏 |
| 品牌与营销 | 品牌知名度, 社交媒体, KOL 策略, 内容营销 |
| 跨境 | 支持的市场, 本地化, 关税/物流, 本地支付方式 |

### `FINTECH` — 金融科技/支付

| 维度 | 关键指标与分析要点 |
|---|---|
| 交易指标 | TPV（总支付量）, 交易笔数, 平均交易额 |
| 资产 | AUM/AUA, 存款增长, 贷款规模 |
| 收入模型 | 交换费, 利差, 订阅, 交易手续费, 浮存收入 |
| 监管 | 持有牌照（按国家）, 监管状态, 合规历史 |
| 风险管理 | 违约率, 欺诈率, 拨备覆盖率, 风控模型 |
| 信任与安全 | 加密, 2FA, 保险覆盖（FDIC/SIPC 等效）, 审计历史 |
| 普惠金融 | 服务不足市场覆盖, 可及性功能 |

### `AI_DEVTOOLS` — AI / 开发者工具

| 维度 | 关键指标与分析要点 |
|---|---|
| 采用度 | 注册开发者数, 月 API 调用量, token 用量, GitHub stars/forks |
| 模型/技术 | 提供的模型, benchmark 表现, 延迟, 上下文窗口, 多模态支持 |
| 定价 | 按 token、按席位、按用量, 免费层限制, 企业定价 |
| 开发者体验 | SDK 语言支持, 文档质量, 首次 API 调用时间, playground/sandbox |
| 生态 | 插件/扩展市场, 集成, 社区贡献 |
| 护城河 | 专有数据, 模型性能差距, 转换成本, 生态锁定 |

### `HARDWARE` — 硬件 + 软件生态

| 维度 | 关键指标与分析要点 |
|---|---|
| 硬件经济 | BOM 成本, ASP, 单台毛利, 制造合作伙伴 |
| 出货量 | 出货台数, 存量装机, 按区域市场份额 |
| 生态 | 软件附加率, 配件附加率, 单台服务收入 |
| 锁定效应 | 专有协议, 内容生态, 数据迁移难度 |
| 供应链 | 关键零部件供应商, 供应链韧性, 交付周期 |
| 售后 | 保修, 维修计划, 翻新, 以旧换新 |

### `HEALTH` — 健康/医疗科技

| 维度 | 关键指标与分析要点 |
|---|---|
| 临床验证 | 证据基础, 临床试验, 同行评审发表 |
| 监管 | FDA 许可/批准, CE 标志, HIPAA 合规, 数据隐私 |
| 成果 | 健康结果指标, 用户报告结果, 护理计划参与度 |
| 支付方集成 | 保险覆盖, 雇主健康计划, HSA/FSA 资格 |
| 专业生态 | 医疗提供者采用, EHR 集成, 转诊网络 |

### `EDTECH` — 教育科技

| 维度 | 关键指标与分析要点 |
|---|---|
| 学习指标 | 完课率, 学习成果, 评估分数, 证书价值 |
| 内容 | 内容库规模, 内容创作模式（自有/UGC/合作）, 更新节奏 |
| 分发 | B2C 直销, B2B2C（通过学校/雇主）, 机构许可 |
| 参与度 | 周学习会话数, 连续签到机制, 社交学习功能 |
| 证书 | 证书/学位认可度, 雇主接受度, 认证 |

### `MEDIA` — 媒体/流媒体/内容

| 维度 | 关键指标与分析要点 |
|---|---|
| 内容 | 内容库规模, 原创占比, 内容投入, 版权成本 |
| 订阅用户 | 总订阅数, 订阅增长, 流失率, ARPU |
| 参与度 | 观看/收听时长, 日均会话数, 内容完成率 |
| 创作者经济 | 创作者数量, 创作者收入, 收入分成模式 |
| 分发 | App, Web, 智能电视, 合作伙伴（捆绑）, 全球可用性 |
