## 1. 执行摘要

**目标：** 让读者在 30 秒内掌握全貌。

写作要求：
- 一段话定位公司（是谁、做什么、为谁做、在行业中处于什么位置）
- 3-5 条关键发现（bullet points），每条用数据支撑
- 一句话总结性评估（机会/风险/值得关注的理由）
- 篇幅控制在 200-300 字

**注意：** 执行摘要最后写，但放在报告最前面。

## 2. 公司概况

**目标：** 建立对公司的基本认知框架。

### 2.1 基本信息表

以表格形式呈现：

| 维度 | 内容 |
|------|------|
| 公司全称 | 中英文名称 |
| 成立时间 | 年份 |
| 总部所在地 | 城市、国家 |
| 创始人/CEO | 姓名及简要背景 |
| 员工规模 | 最新数据 |
| 上市状态 | 股票代码/私有/估值 |
| 官网 | URL |

### 2.2 发展历程

按时间线列出 5-10 个关键里程碑（创立、关键产品发布、重大融资、上市、并购、战略转型等）。

### 2.3 核心管理团队

列出 3-5 位关键高管：姓名、职位、关键背景（教育/前公司/行业经验），重点突出团队的行业相关性和互补性。

### 2.4 企业文化与愿景

一段话概述使命、愿景、核心价值观（如有公开信息）。

**搜索关键词：** `[公司名] founded history`, `[公司名] leadership team`, `[公司名] about us`, `[公司名] crunchbase`

## 3. 业务与产品分析

**目标：** 深入理解公司如何创造价值。

### 3.1 核心产品/服务矩阵

- 列出所有主要产品线/业务单元
- 每个产品：一句话描述 + 目标用户 + 在整体营收中的占比（如有数据）
- 产品之间的协同关系

### 3.2 商业模式解构

- **价值主张**：解决什么核心痛点？为什么客户选择它而非替代方案？
- **收入模式**：订阅 / 一次性销售 / 佣金 / 广告 / Freemium / 使用量计费 / 许可费 — 识别主要和次要收入来源
- **定价策略**：价格区间、定价层级、与竞品的价格对比
- **单位经济模型**：LTV、CAC、LTV/CAC 比率、毛利率（如有数据）

### 3.3 技术能力

- 核心技术栈/专利/技术壁垒
- 研发投入占比
- 技术团队规模和关键技术领导

### 3.4 供应链与运营模式

- 轻资产 vs 重资产
- 关键供应商/合作伙伴依赖
- 运营效率指标

**搜索关键词：** `[公司名] products services`, `[公司名] pricing`, `[公司名] business model`, `[公司名] technology stack patents`

## 4. 市场与行业分析

**目标：** 评估公司所处赛道的吸引力和发展前景。

### 4.1 市场规模

- **TAM**（总可寻址市场）：全球/全行业规模
- **SAM**（可服务市场）：公司当前能触达的市场
- **SOM**（可获得市场）：公司实际能赢得的份额
- 标注数据来源和预测年份

### 4.2 行业趋势与驱动力

- 3-5 个核心行业趋势（技术、监管、消费习惯）
- 行业增长率（CAGR）
- 行业生命周期阶段（萌芽/成长/成熟/衰退）

### 4.3 Porter 五力分析

| 力量 | 分析要点 |
|------|----------|
| 供应商议价能力 | 供应商集中度、替代供应来源、转换成本 |
| 买方议价能力 | 客户集中度、价格敏感度、替代选择 |
| 新进入者威胁 | 进入壁垒（资本、技术、品牌、监管） |
| 替代品威胁 | 替代方案的可用性和性价比 |
| 行业竞争强度 | 竞争者数量、差异化程度、退出壁垒 |

### 4.4 监管环境

- 关键法规和合规要求
- 政策风险与机会

**搜索关键词：** `[行业] market size TAM [当前年份]`, `[行业] industry trends`, `[行业] market report`

## 5. 竞争格局

**目标：** 明确公司在竞争中的位置和优劣势。

### 5.1 竞争对手识别

列出 3-5 个主要竞争对手，每个标注：
- 公司名称 + 一句话描述
- 估值/营收规模
- 核心差异

### 5.2 竞品对比表

以表格形式对比关键维度：

| 维度 | 目标公司 | 竞品A | 竞品B | 竞品C |
|------|----------|-------|-------|-------|
| 核心产品 | | | | |
| 目标客户 | | | | |
| 定价范围 | | | | |
| 营收规模 | | | | |
| 融资/市值 | | | | |
| 核心优势 | | | | |

### 5.3 护城河分析

逐一评估以下竞争优势来源（适用则展开分析，不适用则标注"弱/不适用"）：
- **网络效应**：用户增长是否提升产品价值？
- **规模经济**：规模扩大是否带来成本优势？
- **品牌认知**：品牌是否构成选择偏好？
- **转换成本**：客户迁移到竞品的成本有多高？
- **技术壁垒**：专利/数据/算法是否构成进入壁垒？
- **监管壁垒**：牌照/合规是否构成门槛？

**搜索关键词：** `[公司名] competitors`, `[公司名] vs [竞品名]`, `[公司名] market share`, `[行业] competitive landscape`

## 6. 客户分析

**目标：** 理解公司的客户结构和增长飞轮。

### 6.1 客户细分与画像

- 客户类型：B2B / B2C / B2B2C / B2G
- 核心客户群体画像（行业、规模、地域、决策者角色）
- 客户集中度风险（是否依赖少数大客户）

### 6.2 获客与增长策略

- 核心获客渠道（直销/合作渠道/自然增长/内容营销/产品驱动增长 PLG）
- 增长飞轮：用户增长如何自我强化？
- CAC 趋势（如有数据）

### 6.3 留存与扩展

- 客户留存机制（粘性功能、数据锁定、生态绑定）
- 净收入留存率 NRR / 流失率（如有数据）
- 交叉销售/向上销售策略

**搜索关键词：** `[公司名] customers case studies`, `[公司名] user growth`, `[公司名] go-to-market strategy`

## 7. 财务分析

**目标：** 量化评估公司的财务健康度和增长质量。

### 上市公司

- **营收**：近 3-5 年营收趋势及 YoY 增长率
- **盈利能力**：毛利率、营业利润率、净利润率
- **现金流**：经营性现金流、自由现金流
- **资产负债**：负债率、流动比率
- **市场表现**：市值、P/E、P/S、EV/EBITDA
- **与同行对比**：关键倍数与竞品对比

### 私有/初创公司

- **融资历史**：各轮融资（时间、金额、估值、领投方）
- **估值演变**：估值增长轨迹
- **营收线索**：公开披露的 ARR/MRR/GMV 等指标
- **烧钱率**：如有线索，估算资金跑道
- **核心投资人**：关键机构投资方背景

**搜索关键词：** `[公司名] revenue earnings financial results`, `[公司名] funding rounds valuation crunchbase`, `[公司名] SEC filing 10-K`

---

## 8. 战略与增长

**目标：** 前瞻性分析公司的增长路径。

### 8.1 当前增长引擎

- 核心业务的有机增长驱动力
- 哪些产品线/市场贡献最大增量？

### 8.2 扩张战略

- **地域扩张**：进入新市场/国家
- **产品扩张**：新产品线、平台化
- **并购策略**：近期并购动态及战略意图
- **生态布局**：战略合作、投资布局

### 8.3 战略合作

- 关键合作伙伴及合作性质
- 在产业链/价值链中的定位

**搜索关键词：** `[公司名] strategy growth plan`, `[公司名] acquisitions partnerships`, `[公司名] expansion new markets`

## 9. 风险评估

**目标：** 识别可能影响公司发展的关键风险因素。

按以下维度逐一评估，每项标注风险等级（高/中/低）：

| 风险类型 | 分析要点 |
|----------|----------|
| 市场风险 | 市场增速放缓、需求变化、宏观经济周期 |
| 竞争风险 | 新进入者、巨头入场、价格战 |
| 技术风险 | 技术路线变更、被颠覆的可能性 |
| 监管风险 | 政策变化、合规成本、数据隐私法规 |
| 财务风险 | 现金流压力、融资环境、盈利可持续性 |
| 运营风险 | 关键人才依赖、供应链脆弱性、地缘政治 |
| 客户集中度风险 | 对少数客户/渠道的依赖 |

## 10. SWOT 综合分析

**目标：** 将前述分析收敛为清晰的四象限总结。

以 2x2 矩阵呈现：

|  | 有利因素 | 不利因素 |
|--|----------|----------|
| **内部** | **优势 (S)**：核心竞争力 | **劣势 (W)**：内在短板 |
| **外部** | **机会 (O)**：市场机遇 | **威胁 (T)**：外部风险 |

每个象限列 3-5 条，基于前文分析提炼，不引入新信息。

---

## 11. 结论与展望

**目标：** 给出有立场的总结性评估。

写作要求：
- 对公司整体竞争力的评价（一段话）
- 最值得关注的 2-3 个亮点
- 最需要警惕的 1-2 个风险
- 未来 1-3 年展望（基于当前信息的合理推断）
- 根据用户调研目的给出针对性建议

## 数据质量规范

- **标注来源**：每个关键数据点后标注来源（公司官网/财报/第三方报告/新闻报道）
- **标注时间**：财务数据标注对应财年/季度
- **区分事实与推测**：数据不可得时，用"据估计"、"据行业推算"等前缀
- **交叉验证**：重要数据至少两个独立来源验证
- **信息缺口**：无法获取的信息明确标注"未公开"，而非跳过
