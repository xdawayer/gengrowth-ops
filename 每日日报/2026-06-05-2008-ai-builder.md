1. 今日值得沉淀

1）先拆流程，再配智能体
内容：Peter Yang 认为，知识工作提效的关键不是先换工具，而是先把最痛、最重复的流程拆细，再给 Codex 配 skills 和 integrations，并保留人工 checkpoint。
沉淀价值：先流程化，再智能体化，比空谈全自动更实用。
后续用途：可直接用于梳理 GenGrowth 的内容、研究、交付流程。
来源：https://x.com/petergyang/status/2062740262338929110

2）Claude Code 的质量波动，很多出在产品层
内容：Anthropic 复盘确认，最近体验波动主要来自默认 reasoning effort 调整、长会话上下文被错误清空、以及提示词改动，相关问题已修复或回滚。
沉淀价值：模型能力、算力档位、会话记忆、系统提示必须分开观测。
后续用途：可反查我们自己的 agent 流程是否存在漂移。
来源：https://www.anthropic.com/engineering/april-23-postmortem

3）企业级 Agent 正在形成新架构
内容：Anthropic 两篇更新都在强调同一方向：把 session、harness、sandbox 解耦，把执行环境放回私有沙箱，并通过 MCP tunnels 连接私有工具。
沉淀价值：企业落地重点正从“能不能做”转向“能不能安全进内网并长期运行”。
后续用途：适合沉淀成 GenGrowth 的 Agent 方法论内容。
来源：https://www.anthropic.com/engineering/managed-agents https://claude.com/blog/claude-managed-agents-updates
