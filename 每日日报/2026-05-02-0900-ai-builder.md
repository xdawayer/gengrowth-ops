# AI Builder 早报 - 2026-05-02-0900

> 同步来源：Slack `#ai-builder-daily`；发送时间：2026-05-02 09:00（UTC+8）。

*:robot_face: AI Builders 早间日报 — 2026-05-02*

**:bird: X / Twitter 精选**
- *Andrej Karpathy (AI researcher, 前 Tesla / OpenAI)* — 他在 Sequoia Ascent 分享了对 LLM 新范式的判断：LLM 不只是加速旧工作，而是在催生 menugen、`.md` skills、LLM knowledge bases 和 agent-native economy 这类过去很难成立的新软件形态。[原帖](https://x.com/karpathy/status/2049903821095354523)
- *Swyx (AI Engineer / Latent Space)* — 他认为今年的突破主题是“coding agents breaking containment”：agent 不再只服务程序员，而会进入所有知识工作，并分享了 AI Engineer 团队如何用 agents 支撑每月约 100 万开发者访问。[原帖](https://x.com/swyx/status/2050068468498842058)
- *Cat Wu (Anthropic / Claude Code)* — Claude Security 已进入 public beta，并集成到 Claude Code on the web，可对 repo 做验证过的漏洞发现，并在同一工作流中修复。[原帖](https://x.com/_catwu/status/2049964403177689130)
- *Aaron Levie (Box CEO)* — 他判断 agents 会成为软件最大用户之一，因此所有软件都必须 headless 化，未来商业模式会从单纯 seat 扩展到 API 配额、agent seats 和 consumption pricing。[原帖](https://x.com/levie/status/2050051426446152159)
- *Ryo Lu (Cursor Design)* — 他提出“no opinion is opinionated”：真正通用的产品不是没有立场，而是在核心概念、抽象和可扩展性上有强一致观点。[原帖](https://x.com/ryolu_/status/2049866003287576978)
- *Garry Tan (Y Combinator President & CEO)* — 他推荐在 Karpathy-style Knowledge Wiki 上叠加 MIT-licensed、开源的 GBrain，尤其适合 Hermes Agent 和 OpenClaw 这类 personal AI 场景。[原帖](https://x.com/garrytan/status/2050096324100682097)

**:studio_microphone: 播客**
- *Training Data / Demis Hassabis* — Demis Hassabis 回顾 DeepMind 从游戏、neuroscience、reinforcement learning 到 AlphaFold 的路径，核心观点是 AGI 应先被打造为“极其智能、精确的工具”，再去处理 agency、consciousness 等更深问题。[Demis Hassabis on Building DeepMind, AlphaFold, and the Final Stretch to AGI](https://www.youtube.com/playlist?list=PLOhHNjZItNnMm5tdW61JpnyxeYH5NDDx8)

**:memo: 博客**
- *Claude Blog* — Claude Code desktop 重新设计为 parallel agents 工作流：新增多 session sidebar、拖拽布局、内置 terminal 和 file editor、更快 diff viewer、插件 parity、SSH 支持和 usage 视图，面向“同时跑多个 agent 任务并人工编排”的开发方式。[Redesigning Claude Code on desktop for parallel agents](https://claude.com/blog/claude-code-desktop-redesign)
