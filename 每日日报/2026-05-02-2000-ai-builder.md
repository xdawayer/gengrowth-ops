- *Amjad Masad (Replit CEO)* — Replit 十周年，Amjad 宣布从太平洋时间早上 5 点开始免费开放 24 小时，并回顾 Replit 从 2011 年开始推动“让所有人都能 coding”的路径。[原帖](https://x.com/amasad/status/2050479551537619413)
- *Claude (Anthropic AI assistant)* — Claude 宣布开发者大会 Code with Claude 下周回归，面向 Claude Code 新手和已有开发者开放 livestream 注册。[原帖](https://x.com/claudeai/status/2050252933866930339)
