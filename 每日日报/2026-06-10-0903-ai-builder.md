1. Kafka 整理了 Anthropic Academy 课程《Introduction to agent skills》第一章，核心意思很明确：如果你反复向 Claude 解释同一件事，这件事就应该被沉淀成一个 skill。
2. skills 的关键价值不是“多一个配置文件”，而是按需触发、节省上下文、支持个人全局复用与项目级团队共享；它与 CLAUDE.md、hooks、subagents 处在不同层。
3. Anthropic 官方课程页还明确把 skills 往更完整的工程体系推进：multi-file skills、allowed-tools、团队分发、enterprise managed settings、troubleshooting。
4. 对 GenGrowth 的意义：这条内容正好补上我们已有的 Dynamic Workflows / Loop Engineering / 基座+Skills 体系里“可复用能力单元”这一层，说明 skill 不是 prompt 附件，而是 agent 基础设施。
